并网逆变器的硬件拓扑与滤波器设计原理

我们在做大功率并网逆变器开发时，绝大多数方案本质上都是电压源型逆变器（VSC）。我们在直流侧挂上大容量的母线电容来稳住直流电压，然后通过高频 PWM 调制，在交流侧输出高频方波。

这里我得和大家分享一个核心逻辑：电网是个非常庞大、内阻极低的“硬”电压源。如果把逆变器这端的电压源直接往电网上接，哪怕只有一丝丝的相位差或者幅值抖动，回路里的极低阻抗就会导致几安培甚至几十安培的瞬间环流，结果就是直接触发过流保护，严重时直接烧管子。所以，我们在硬件设计上必须在中间串联高阻抗元件来做物理缓冲。

并网点电压是被电网死死钳住的，逆变器想把功率输送进去，核心逻辑其实是“用电流源去对接电压源”。我们在控制算法里要把逆变器模拟成一个高阻抗的交流电流源。这样电网就成了天然的钳位器，帮我们吸掉电流纹波。而 PWM 输出的高频方波电压，就得靠滤波器来滤成平滑的并网电流了。

在滤波器选型上，我们通常在 L 和 LCL 之间做权衡：

滤波器类型 结构与参数特性 物理逻辑与控制意义
L 滤波器 单纯依赖电感进行滤波 电感直接抗拒电流突变，将 PWM 矩形波积分成三角波电流。接入电感后，从电网侧观察逆变器具备了高输出阻抗，物理表象转化为纯粹的电流源承担者。
LCL 滤波器 引入极小容量的并联电容并配合前级与后级电感 逆变侧电感承担电流源的主要角色。极小的电容为高频开关纹波提供低阻抗通路，带来极强的纹波衰减能力。此电容主要吸收开关纹波，不可用于储能维持电压。若容量过大不仅产生巨大工频无功电流，还会降低谐振频率使其落入控制带宽引起系统失稳。

跟网型与构网型控制逻辑对比

逆变器的控制架构现在分成了两条大路：跟网型（Grid-following）和构网型（Grid-forming）。我们在研发中经常需要根据电网的实际情况来选择：

特性维度 跟网型 构网型
模拟对象 受控电流源 带虚拟阻抗的软电压源
同步机制 依赖 PLL 捕捉电网相位与频率 不依赖 PLL，主动支撑频率与电压
控制内环 dq 坐标系下的电流控制，利用 PI 调节器跟踪参考值 引入戴维南等效电路，模拟内部电压源与虚拟阻抗
运行状态 恒流运行，向电网泵入功率 动态调节功角，实现平稳可控的功率交互
异常响应 失去电网参考或触发 OVP 或 UVP 时迅速停机 具备下垂控制，主动退让并支撑电网
适用场景 常规并网环境，依赖强电网支撑 微电网、高比例可再生能源系统、弱电网或独立运行

构网型逆变器的并网机制与瞬态支撑原理

直觉上，把构网型逆变器作为一个“电压源”和电网并联，好像违背了“双电压源不能直连”的电力电子第一性原理，弄不好会有毁灭性的瞬态冲击。但实际上，我们在设计中是通过物理硬件、控制算法和能量传输这三个维度的配合，把构网型逆变器做成了一个带内阻的“软电压源”，从而安全并网：

  • 物理层面的阻抗隔离：逆变器桥臂输出和并网点之间有物理电感隔着。LCL 滤波器里的逆变桥、逆变侧电感 和滤波电容 C 拼起来，相当于在内部做了一个干净的正弦电压源（内部电压参考点 E）。并网侧电感 则充当安全防护隔离带，不让内部电压和外边电网硬碰硬。
  • 引入虚拟阻抗：我们在控制环路里采样输出电流并做负反馈，相当于给系统塞进去了一个等效的复阻抗 。这样整个系统就成了戴维南等效模型（电压源串联物理阻抗加虚拟阻抗）。电网电压一波动，总阻抗就能顶住这个压差冲击，化解成平缓的电流吞吐。
  • 下垂控制的主动退让：构网型不硬顶着额定频率和电压不放。有功大时主动把频率往下垂一垂，无功大时主动把电压往下垂一垂，从而实现柔性互动。

  • 硬件级的瞬态物理支撑（防炸管底牌）：这是构网型最牛的地方。当电网发生短路等突发故障时，跟网型逆变器的软件算法要花几十毫秒才能算出应对策略，这段延迟容易导致器件过流炸管。而构网型因为它的滤波电容 C 是并联在回路中的物理电压源，跟电网中间只隔着并网侧电感 ，所以在跌落发生的一瞬间（几微秒内），算法甚至还没反应过来，电容和电感就会本能地输出巨大的无功电流去撑住电网。这就是实打实的物理惯量支撑。

逆变器基础

逆变器的控制核心大多走数字控制路线，拓扑里开关管一大堆。直流侧的电容主要用来平衡无功能量的吞吐，并在充放电过程中维持直流母线电压的稳定。

并联逆变器环流

多机并联运行是我们在工程中常碰到的工况，但也是个大坑，很容易出环流。环流是指电流直接在并联的逆变器之间循环流动，根本不往负载去，平白无故增加发热甚至烧机器。

导致环流的根子主要在于各台逆变器之间的参数没有对齐，常见避坑点有：

  • 输出电压幅值不一致
  • 触发相位不一致
  • 频率漂移不同步
  • 输出阻抗不对称

逆变器控制目标与策略

逆变器的控制逻辑主要看是用在什么场景，并网和离网的设计重点完全不同：

并网逆变器：电流环是绝对核心

  • 控制目标：由于电网的电压和频率是极硬的，逆变器根本无法改变它们，所以并网逆变器的任务是朝着这个已知的电压源里，注入波形干净、幅值相位精准可控的电流。
  • 功率控制方式：
    • 有功功率 P：控制注入电流中与电网电压同相的直流分量 。
    • 无功功率 Q：控制与电网电压正交（超前 90°）的交流分量 。
  • 控制环路：主要做电流环。
  • 电压采样的作用：并网虽然不控电压，但必须采电压，目的是：
    • 锁相与同步：给锁相环（PLL）提供参考，抓取电网的实时频率和相位，这是 Park 变换的基准。PLL 要是没锁住，并网就是灾难。
    • 前馈解耦：把采样到的电网电压直接作为前馈引入控制，抗住电网波动扰动，提高动态响应。

离网逆变器：电压环是绝对核心

  • 控制目标：在孤岛模式下，逆变器自己就是老大，必须为负载生出一个稳定、干净的电压。
  • 控制环路：一般用电压外环加电流内环的双闭环结构。
    • 电压外环：稳定输出正弦波电压（如 220V/50Hz）。
    • 电流内环：快速跟踪外环指令，并做限流保护以提高抗冲击能力。
  • 同步参考：不需要 PLL，自己内部定时器给频率和相位基准即可。

单相全桥逆变器

很多初学者觉得逆变器和 DC-DC 是两码事，但实际上，单相逆变器的一个桥臂（上下管互补）在动态上完全等效于一个双向的 Buck-Boost 变换器。以感性负载（如电机驱动）为例：

  • 等效 Buck 模式（能量从直流向交流跑）：电机加速时，我们需要输出正弦电压推电流进电机。此时上管进行高频 PWM 斩波，把直流母线电压降压后输出给电机，这过程和 Buck 一模一样。
  • 等效 Boost 模式（能量从交流回馈直流）：电机减速制动时，电机相当于发电机，产生反电动势。此时我们控制下管进行高频斩波，利用电感的储能效应，把较低的反电动势升压泵回直流母线电容，这过程完美符合 Boost 升压回馈。

逆变器这种天生具备双向流动能量的能力，使它在面对阻性负载时像 Buck，面对感性有源负载时则在 Buck 和 Boost 间高频交替。

    • 三相四线制第四桥臂：在输出端加第四个桥臂来接零线，主动控制零线电流来对冲中点电位不平衡。

三电平逆变器

常用的三电平逆变器主要有以下几种拓扑结构：

  • 传统三电平 NPC（I-Type NPC）：每个桥臂由四个功率管串联组成，中点通过两个钳位二极管连到直流中性点。因为形状像“I”，所以叫 I 型。缺点是损耗分布极其不均，外侧管子主要分担开关损耗，内侧主要分担导通损耗，导致内侧管子发热严重成为瓶颈。
  • T型三电平 NPC（T-NPC）：标准半桥加一个连到中点的双向开关（两个背靠背的开关管）。因为形状像“T”，所以叫 T 型。优势是器件少（省了两个二极管），在输出零电平时只经过一个双向开关，路径短导通损耗极低，光伏逆变器里用得很普遍。缺点是中点管子承受的开关应力大，而且主开关需要抗全母线电压。
  • 有源中点钳位 NPC（ANPC）：为了解决 I 型 NPC 损耗不均的问题，把两个钳位二极管换成 active 的功率开关管。这样在控制上可以通过调整开关组合，主动把损耗在内外管之间调配实现热均衡，大幅提升功率密度，但代价是驱动和控制复杂，成本也高。

T-NPC控制策略

一个标准的 T 型三电平桥臂接线很明确：外侧 S1 接正母线，S4 接负母线；内侧 S2 和 S3 背靠背组成双向开关接直流中点 N。

根据各开关管的动作，输出如下电平：

输出电平 S1 状态 S4 状态 S2 状态 S3 状态 电流路径
正电压 ON OFF OFF OFF 电流从正母线经S1流向输出端
零电压 OFF OFF ON ON 电流在输出端与中点N之间经S2/S3双向流动
负电压 OFF ON OFF OFF 电流从负母线经S4流向输出端

在时序控制上，我有几点调试避坑经验：

  • 外侧的 S1 和 S4 属于严格的互补对，一定要留足死区时间，否则直流正负母线直通就是一声巨响直接炸管。
  • 正半周调制时：S4 保持关断，S1 和 (S2, S3) 互补动作。
  • 负半周调制时：S1 保持关断，S4 和 (S2, S3) 互补动作。
  • 切换时序：必须遵循“先关后开”原则。比如从零电平切到正电平，必须先关断 S2 和 S3，跑完死区时间后，再开通 S1。

PWM 合成交流波形的基本原理

我们搞数字控制，是用高频的开关周期（比如 10kHz）去逐点描绘出低频的工频周期（如 50Hz，20ms）。

以 10kHz 开关频率做 50Hz 正弦波为例，一个工频周期内有 200 个开关周期。每个周期控制器都会重新计算一次当前目标正弦值：

  • 起始位置（第 1 个开关周期）：正弦参考值接近 0（如 +0.03），控制器走正半周策略，S1 只导通 3% 的时间，S2/S3 导通 97%。
  • 正半周峰值（第 50 个开关周期）：参考值达到 +1.0，S1 几乎 100% 导通。
  • 负半周起始（第 101 个开关周期）：参考值越过零点（如 -0.03），控制器切换至负半周策略，S4 导通 3%，S2/S3 导通 97%。

从 SPWM 到 SVPWM

在三相系统里，SVPWM 是最常用的。跟一维的 SPWM 相比，SVPWM 是在二维空间里去合成一个旋转的电压矢量：

特性 SPWM (单相) SVPWM (三相)
控制目标 单个正弦电压值 旋转的空间矢量
判断依据 V_mod 的正负号 V_ref 的角度 (位于哪个扇区)
周期概念 清晰的正/负半周 6个扇区的循环
拼凑方式 用正/负电压与零电压拼凑 用相邻的两个有效矢量和零矢量拼凑
最终结果 一路正弦波 一个旋转磁场 (三路相位差120°的正弦波)

研讨与答疑

在实际研发和 PLECS 仿真中，常有兄弟提到下面这几个非常硬核的工程问题，这里我统一给大家解答一下：

Q1：为什么三电平中性点支路要用源极对源极连接（Common Source），漏极对漏极（Common Drain）不行吗？

解答： 在仿真里，这两种接法其实都能跑通。但我们在实际硬件开发中，几乎全部采用源极对源极的接法。
原因主要在于驱动电路的设计难度：

  • 源极对源极（Common Source）：两个管子的源极接在一起。因为源极（发射极）是驱动电压的参考地（Reference GND），把源极接在一起，这两个管子就可以共用一组隔离驱动电源的参考地。这在硬件上省下了一大块电源设计成本。
  • 漏极对漏极（Common Drain）：源极分别朝向两侧，电位是悬空的。这就意味着你需要两组完全隔离、毫无关系的驱动电源，无形中在硬件上给自己找了巨大的麻烦。
  • 封装便利：市面上的 T 型三电平 IGBT 模块（英飞凌或三菱等），内部也全都是发射极对发射极封装。

  • 线电压 （A、B 相电感前输出相减）：波形应是标准的 5 电平 PWM。
  • 负载相电压 （电感前对负载中点 n）：波形看起来像三电平，但平顶会有很多高频缺口或锯齿。这是正常的，因为负载中点 n 和直流中点 O 之间没接，共模电压 抖动叠加在了波形上。
