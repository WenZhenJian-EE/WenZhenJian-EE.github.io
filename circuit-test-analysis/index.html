对于常规 Buck 开关电源，交流耦合测量下的稳态纹波峰峰值在 30mV 以内通常被认为是优秀水平。然而在数字电源调试中，仅看绝对电压幅度是不够的，波形的包络形态往往暴露出控制环路的潜在隐患。以下是对波形异常现象的深度解析与优化策略。

示波器测量规范与高频振铃分析

标准电源纹波测试需要采用严格的测量手法以消除外界干扰信号。以下为推荐的标准测试参数设置组合及其核心作用：

测试参数 原理与作用
探头 x1 档位 不经过示波器内部额外放大，信噪比最佳，避免引入 x10 档位下的底噪
接地弹簧 极大减小地线环路面积，避免引入空间的高频辐射干扰，优于长带线鳄鱼夹
20MHz 带宽限制 有效滤除与电源工作无关的高频环境白噪声干扰
交流耦合测量 滤除直流分量，专门用于放大并观察微小的交流波动成分

在时基较大的情况下，波形上呈现的密集尖峰容易被误判。将时基展开至 50us 级别时，可以清晰观察到这些尖峰的本质并非电感充放电波形。

  • 现象本质：这是 MOSFET 在导通或关断瞬间的高 与高 变化，与 PCB 寄生电感及 MOSFET 结电容发生高频谐振产生的开关瞬态振铃现象，频率通常在几十至上百兆赫兹。
  • 影响评估：此类尖峰幅值极小，通常不足以穿透芯片引脚的寄生电容，对常规数字电路或功率级供电的功能无实质影响。
  • 解决思路：控制代码无法消除该现象，需要通过优化 PCB 布局以减小开关环路面积，或在硬件电路上增加 RC 吸收电路来解决。

低频包络起伏与极限环振荡机理

理想的模拟闭环电源由于误差放大器输出连续的模拟电压，其稳态波形呈现为由电感充放电引起的纯高频三角波，即极其规律的开关纹波。但在数字电源中，高频纹波上往往会叠加低频包络起伏。

  • 动态响应失效与震荡的区别：如果在额定负载下输出电压发生大幅突变或跌落，属于动态响应失效或调节能力丧失。而电压维持在目标值附近但出现低频起伏，属于控制环路相位裕度或增益裕度不足导致的轻微震荡。
  • 极限环振荡现象：在宏观时间轴上，高频开关纹波叠加在一个低频的正弦波包络上，表现为波形上下沿周期性地变粗、变细与偏移。
  • 核心成因：数字电源的 PWM 由单片机内部定时器产生，占空比呈离散阶梯状。当目标输出电压对应的理想占空比落在两个离散状态之间时，由于 ADC 采样精度或 DPWM 分辨率不足，PID 算法无法找到精确的稳态落脚点。系统被迫在相邻的两个占空比大台阶之间来回横跳，产生量化噪声。这种占空比的周期性交替反映在输出电容上，即表现为宏观的低频包络。

基于底层时钟与 PWM 分辨率的优化策略

解决极限环振荡的根源在于提升系统调节的量化粒度，即最大限度提高数字电源的控制精度。

  • 基础时钟配置原则：占空比的本质计算公式为 。如果 PWM 模块开启了时钟分频，定时器基础时钟变慢，在维持相同开关频率的前提下，周期寄存器的最大计数值变小，这导致占空比的最小分辨率步进变大。因此，应关闭 PWM 模块的时钟分频，使其运行在最高系统主频，以最大化周期寄存器值，从而将占空比切分至最细。
  • HRPWM 的应用：对于具备相关外设的 DSP 主控，应开启 HRPWM 模块。通过微边沿定位技术，将占空比的调节步进细化至皮秒级别。极高的分辨率使输出能够无限逼近理想占空比，有效消除占空比抖动，使输出波形恢复为平滑的直线。

控制环路 PI 参数调节与权衡验证

在硬件分辨率调至最优后，若仍存在轻微包络，可通过调节 PI 环路参数进行修整，在响应速度与鲁棒性之间寻求平衡。

  • 参数调节方向：适当减小比例增益 ，使环路响应变缓和，避免剧烈纠正微小误差；适当增加积分时间 ，降低积分作用，防止系统在目标值附近因响应过快而产生过冲与反向修正。
  • 性能牺牲代价：趋于保守的 PI 参数设置会牺牲系统的暂态响应性能。在负载发生突变跳变时，输出电压的跌落幅度会增加，且电压恢复时间会变长。
  • 标准调试工作流：首先检视并优化 PWM 时钟分频配置以夯实基础分辨率；其次微调 PI 参数观察稳态包络的平滑改善情况；最后必须进行负载瞬态跳变测试。若负载跳变时电压跌落超出规格限制，说明 PI 参数设置过于偏软，必须返回第一步继续压榨 PWM 分频与分辨率性能，切忌因掩盖硬件配置缺陷而牺牲系统的动态响应指标。
