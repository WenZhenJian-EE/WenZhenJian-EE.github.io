开关电源模拟控制模式与环路架构设计手记

模拟环路底层机理：并联竞争二极管选择、串级峰值电流模式、COT 滞回响应与软硬件控制分野

在电力电子研发中，控制环路是整个开关电源的“大脑”。虽然现代数字控制（MCU/DSP）大行其道，但模拟控制芯片（如经典的 UC3842、TL494、UCC28C42）凭借其零延迟、无限分辨率和高可靠性，依然是反激辅助电源、电池充电器和主板 VRM 模块的首选。

今天这篇笔记，我们深入探讨并联竞争与串级主从控制的硬件电路设计细节、各种模拟控制模式的物理优缺点，并梳理模拟与数字控制在工程落地时的选型博弈。

一、 并联竞争控制：CC / CV 的无缝二极管二选一物理电路

在设计实验室可调恒压恒流源、电池充电器（充电初期恒流 CC，充满前转恒压 CV）时，我们常用并联竞争控制架构。其核心逻辑是：电压环和电流环各自计算占空比，最保守的那个环路（即要求占空比最小的环路）胜出，接管控制权。

1. 硬件二选一电路的物理本质（以二极管并联为例）

在经典控制器 TL494 或分立运放环路中，电压误差放大器（EA1）和电流误差放大器（EA2）的输出端，通常通过两个反向二极管并联接在一起，输出到 PWM 比较器的负端（反馈引脚 COMP）：

  • 工作机制：电压环和电流环的运放输出较高的控制电位代表要求输出更小的占空比（对非反相 PWM 而言）。
    当电池电压未满时，电压环输出低电平（表示占空比可以很大），而电流环因达到恒流设定点输出高电平。由于 D2 导通，D1 反偏截止，电流环的高电平强行压过电压环，夺取 COMP 点控制权，维持恒流。
    一旦电池充满电压超标，电压环运放输出电位飙升，D1 导通，D2 截止，电压环夺走控制权，无缝切换为恒压模式。
  • 设计避坑：两个竞争二极管必须选用漏电流极小、开关速度极快的肖特基管或快恢复管（如 BAV99），否则在几十 kHz 以上开关频率下，二极管反向恢复电流及漏电会引起 COMP 信号畸变，导致开关脉冲抖动。

二、 串级控制：峰值电流模式（Peak Current Mode）主从双环

串级控制（Cascade Control）是电压外环与电流内环构成的分层主从结构，以 UCC28C42、UC3843 系列芯片为最典型代表。

1. 主从互锁机制

  • 电压外环（主控）：慢速环路。根据输出电压与基准的偏差，算出当前系统需要多大的电感电流，输出一个动态的电压上限（设定内环的参考基准 ）。
  • 电流内环（从属）：极速环路。逐个周期（Cycle-by-Cycle）采样电感电流峰值。当检测到电流斜坡达到外环设定的限流基准瞬间，立即关断 PWM 开关管。
  • 短路保护竞争：在 UCC28C42 芯片中，外环输出的控制电平被芯片内部限流基准（通常为固定 1.0V）强行限制。如果输出发生短路，电压跌落导致外环索要无穷大电流，但 1.0V 硬件基准取得最高控制权，直接进入逐周期 1V 硬关断保护状态，保护功率管免受冲击。

2. 串级控制的三大物理优势

  1. 极快响应（输入电压前馈）：当输入电压突变时，电感电流斜坡率 瞬间改变。电流内环在几百纳秒内就会自动调整占空比，完全不需要等待电压外环反应。
  2. 环路易补偿：电流内环强行将电感“转换为”一个受控电流源，把 LC 滤波的双极点谐振衰减系统降阶为单一的电容一阶系统，补偿极易设计。
  3. 逐周期硬限流：内环每一个开关周期都监测峰值电流，安全 SOA 边界被死死锁住。

三、 几种主流模拟控制模式的区别与定位

除了上述两种架构，模拟控制在不同赛道有清晰的划分：

  • 单电压控制模式（早期传统方案）：只采输出电压，通过与锯齿波比较产生 PWM。抗噪性极佳（不引入电流采样毛刺），但由于没有电流内环，无法逐周期限流，在大占空比（）下极易导致磁芯偏磁饱和，现多用于小功率或固定负载设计。
  • 滞回控制 / COT 恒定导通时间控制（瞬态之王）：没有固定时钟，电容电压跌破 VREF 瞬间触发固定导通时间 的脉冲。瞬态响应在纳秒级，无环路带宽限制。广泛用于主板 CPU 供电 VRM，但由于开关频率随负载和电压变化，EMI 滤波器极难设计。
  • 变频和谐振控制（FM, 如 LLC 变流器）：占空比固定为 50% 附近，通过改变开关频率来调节谐振腔阻抗以控制输出增益。利用谐振过程实现功率管零电压开通（ZVS）和零电流关断（ZCS），是高效大功率模块的绝对主流。

四、 模拟与数字控制在工程中的选型底线

在研发 30kW 及以上大功率系统时，控制器的选择并不是一成不变的，它们有明确的设计边界：

选型对比维度 模拟控制芯片 (如 UCC28C42) 数字控制芯片 (如 C2000 DSP / STM32G4)
控制机理 外部阻容物理搭建 PID，求解微分方程。 ADC 分时采样，计算软件差分 PID 差分方程。
动态延迟 零延迟，比较器和运放连续运行。 存在 ADC 采样保持延迟、DSP 指令计算延迟。
分辨率 无限分辨率，无量化噪声。 受限，比如 12 位 ADC 带来约 0.8mV 量化噪声。
调试成本 复杂。需要反复烙铁焊接更改阻容元件（COMP 脚）。 极低。直接在线修改软件代码的 参数。
复杂逻辑 极差。无法处理复杂的切相、自动变频及通讯功能。 极佳。易于实现复杂的时序协同、CAN 监控和非线性控制。
  • 工程选型法则：
    1. 辅助小电源：反激供电等，一律选用模拟芯片。不占用 CPU 算力，上电瞬间启动，死机不炸管，成本低且极度可靠。
    2. 主功率变流级（LLC / PFC / 储能 PCS）：一律选用数字 DSP。它们需要高精度的切相、变频效率优化、软件保护阈值下发以及 CAN 状态通讯。
    3. 极高速瞬态负载（CPU 核心 VRM 供电）：一律选用模拟滞回/混合芯片。因为数字 ADC 采样和指令延迟无法跟上 CPU 纳秒级大电流跃变。
