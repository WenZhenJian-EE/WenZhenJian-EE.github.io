电源控制架构核心解析：并联竞争与串级控制

电源控制架构核心解析：并联竞争与串级控制

并联竞争控制
并联竞争控制是一种多个控制环路平行运行、相互竞争控制权的架构。其核心逻辑通常遵循最保守者优先的原则，即要求最严格的环路将获得最终的控制权。

  • 电路结构：以 TL494、SG3524 等芯片为典型代表。这类芯片内部集成两个或多个功能相同的误差放大器，其输出端通过二极管或门结构或最小值选择器连接至同一个 PWM 比较器的输入端。
  • 工作机制：电压环和电流环等多个环路同时独立运行，各自计算出为满足其控制目标所需的占空比。PWM 决策机制会选择所有环路中要求占空比最小（即输出误差电压最高或限制性最强）的那个作为最终的执行指令。例如，若电压环要求 50% 的占空比，而电流环要求 20% 的占空比，系统将采纳电流环的指令，以 20% 的占空比运行。
  • 典型应用：该架构非常适用于需要实现不同工作模式无缝切换的场景。
  • 电池充电器：充电初期，电流环接管以实现恒流充电；当电池电压接近充满时，电压环接管以实现恒压充电。
  • LED 驱动电源：正常工作时，电流环接管以恒定电流点亮 LED；当 LED 发生断路导致输出电压异常升高时，电压环介入，实现过压钳位保护。
  • 实验室可调电源：在恒压和恒流模式之间自动切换。

串级控制
串级控制采用分层的内外环结构，外环的输出作为内环的参考目标，形成主从关系。

  • 电路结构：以 UCC28C4x 和 UC384x 系列芯片所代表的峰值电流模式控制最为典型。这类系统中通常只有一个电压误差放大器作为外环，其输出直接设定电流比较器的参考阈值，构成内环。
  • 工作机制：外环（电压环）作为指令发出者，根据输出电压与基准的偏差，实时设定内环（电流环）需要达到的目标值，通常是电感电流的峰值。内环作为执行者，其唯一任务是精确地跟踪并执行外环下达的指令。电流环本身没有独立的固定参考，完全听命于电压环。
  • 竞争特例：在 UCC28C4x 这类芯片中，唯一的竞争逻辑发生在正常工作与硬件过流保护之间。电压环输出的正常电流阈值与芯片内部固化的 1V 固定限流基准进行竞争，PWM 比较器会选择两者中的较小值作为最终的关断阈值。当发生短路时，电压环可能因输出电压骤降而要求极高的电流，此时 1V 的内部基准将夺取控制权，强制系统进入逐周期限流保护状态。
  • 典型优势：
  • 极快的动态响应：具备输入电压前馈特性，能快速响应输入电压的变化。
  • 易于环路补偿：电流内环将电感极点移至高频，使整个功率级在电压外环看来简化为一阶系统，补偿设计变得简单。
  • 逐周期限流保护：内环逐个周期监测电流，提供快速可靠的过流保护。
  • 典型应用：反激式辅助电源、高性能稳压源（如为 CPU 或 FPGA 供电的降压电路）、光伏逆变器与功率因数校正（PFC）电路。

其他主流模拟控制模式及应用场景

单电压模式

  • 核心原理：系统仅包含一个电压控制环路。将采样得到的输出电压与基准电压进行比较，产生的误差信号直接与一个固定频率的三角波或锯齿波进行比较，以生成 PWM 驱动信号。该模式不采样电流信号参与环路控制，电流采样通常仅用于触发独立的短路保护逻辑。
  • 典型特征：由于不引入可能含噪声的电流采样波形进入主控制环路，其抗噪声性能极佳，且占空比调节范围较宽。
  • 典型应用：早期的半桥或全桥转换器，如旧式 PC 电源。在这些拓扑中，使用电压模式可以有效避免峰值电流模式在占空比大于 50% 时可能引发的磁通不平衡与变压器饱和问题。

滞回控制与纹波控制

  • 核心原理：系统内部没有误差放大器和固定频率的时钟振荡器。其开关动作由输出电压纹波直接触发。当输出电压跌落至预设的下限阈值时，开关管导通；当电压回升至上限阈值时，开关管关断。其中，导通时间固定的称为 COT (Constant On-Time) 控制。
  • 典型特征：瞬态响应速度极快，可达纳秒级别，因为无需等待补偿环路的响应。但其开关频率会随输入电压和负载电流的变化而变化，给 EMI 滤波器的设计带来挑战。
  • 典型应用：对瞬态响应要求极高的场合，如主板上的处理器核心供电 VRM 模块和手机内部的直流变换器。

变频控制

  • 核心原理：该模式不调节占空比，而是通过调节开关频率来控制功率传输。它利用谐振网络（通常由电感和电容构成）的阻抗随频率变化的特性来调节系统的电压增益。
  • 典型特征：能够实现零电压开通 (ZVS) 或零电流关断 (ZCS) 等软开关技术，极大地降低了开关损耗，从而实现非常高的系统效率。
  • 典型应用：大功率、高效率的电源，如 LLC 谐振变换器，广泛用于服务器电源、笔记本适配器和高功率手机快充头。

模拟控制与数字控制的维度对比

创造力与实现载体

  • 数字控制：以 MCU 或 DSP 为载体，控制逻辑由软件代码定义。这赋予了其极高的灵活性和创造力，可以轻松实现模型预测控制、自适应 PID 调节以及基于复杂状态判断的非线性控制策略。其代价是存在算力成本、ADC 采样延迟、量化误差以及软件运行的潜在风险。
  • 模拟控制：以专用的模拟控制芯片（如 UCC28C4x）为载体，控制逻辑由芯片内部的硬件拓扑、外部的 PCB 走线及阻容参数共同定义。通过在补偿引脚搭建 Type II 或 Type III 等网络，实质上是在构建特定的微分积分方程来合成所需的传递函数。模拟控制具备无限分辨率和零延迟的特性，连续的模拟信号使得比较器能够瞬时感知微小的电压变化。

模拟芯片的设计局限与工程发挥空间

  • 幸存者偏差：学术界提出了成百上千种控制策略，但由于模拟集成电路设计对鲁棒性、低成本（需用少量晶体管实现）和通用性的严苛要求，最终被市场广泛接受并大规模量产的仅有电压模式、峰值电流模式和滞回控制等少数几类。更复杂的控制算法则交由数字控制实现。
  • 硬件算法注入：尽管模拟芯片的内部逻辑是固定的，但工程师仍能通过巧妙设计外围电路来拓展其控制维度，实现“硬件编程”。
  • 斜率补偿：在电流采样引脚上叠加一部分来自振荡器的锯齿波信号，以改变电流斜坡的判定斜率，从而解决峰值电流模式在占空比大于 50% 时可能出现的次谐波振荡问题。
  • 前馈控制：从输入电压端通过电阻连接至振荡器引脚，通过改变振荡频率或斜率来提前响应输入电压的波动，实现前馈控制。
  • 软启动与软关断：利用二极管和电容网络对补偿引脚的电压进行钳位和充放电控制，从而人为地控制系统环路建立的速度，实现平稳启动或关断。

应用场景选型指南

应用场景 推荐控制方案 选型原因
辅助电源 模拟芯片 成本极低，外围电路简单。将复杂的物理层控制逻辑固化在硅片中，运行稳定可靠，无软件死机风险，是解决基础供电需求的最优解。
复杂大功率电源（如 PFC 结合 LLC） 数字 MCU 擅长处理复杂的时序配合、多重保护逻辑、系统通信协议以及切相、变频等高阶效率优化算法，能够实现对整个电源系统的智能化管理。
处理器核心供电 专用模拟或混合芯片 处理器负载跳变速率极快，数字控制的 ADC 采样延迟无法满足纳秒级的响应需求。必须采用响应速度极致的滞回控制或专用的多相模拟/混合架构。
