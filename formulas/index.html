拓扑占空比与输出电压关系

在电力电子技术中，理解占空比 与输出电压 的关系是电路分析与控制算法设计的核心。输出电压通常与占空比呈正相关或负相关，其核心逻辑建立在伏秒平衡原则之上。

关键更新：影响转换比的进阶因素

CCM 与 DCM 的区别

上述公式仅适用于连续导通模式 CCM。当负载变轻进入断续导通模式 DCM 时：

  • 电压升高：在相同占空比下，DCM 的输出电压通常比 CCM 高。
  • 负载相关：在 DCM 下，输出电压不仅取决于 ，还与负载电流、电感量、开关频率直接相关。

谐振拓扑 LLC 的不同逻辑

对于现代高效率应用中常见的 LLC 谐振变换器：

  • 频率控制 PFM：它的输出电压主要通过调节开关频率 而非占空比来控制。
  • 占空比：通常固定在接近 （扣除死区时间），以实现初级开关管的零电压开关 ZVS。

逆变器 DC-AC 特性

  • 调制指数 ：在 SPWM 或 SVPWM 控制中， 代替了固定占空比的概念。
  • 关系： (半桥) 或 (全桥)。输出电压有效值与 呈线性正相关。

实际工程中的偏差与非理想因素

在工程实践中，理想公式 会因为以下因素产生偏差：

  • 二极管压降 ：在低压输出应用中（如 5V 或 3.3V），二极管的 压降会显著改变转换比。
  • 导通损耗 RDS(on) 与 ESR：功率管、电感电阻上的压降会随电流增大而增加，导致输出电压随负载增加而下降（负载调整率）。
  • 死区时间：在桥式拓扑中，死区时间会“削减”有效占空比，导致实际输出电压略低于理论值。
  • 磁芯饱和与寄生参数：变压器漏感会引起电压尖峰，影响占空比的有效利用率。

控制策略：从理论到闭环

鉴于非理想因素的复杂性，工程上很少采用纯开环控制。

闭环反馈机制

  • 采样与反馈：实时监测输出电压 。
  • 误差计算：与参考电压 对比，生成误差信号。
  • PI/PID 调节：补偿器根据误差自动增减占空比 。如果负载增加导致电压下跌，PI 调节器会自动增大 以补偿内阻压降。
  • 动态补偿：现代控制算法（如电压前馈控制）可以针对 的波动瞬间调整 ，从而保持输出稳定。

总结

理论公式定方向，闭环控制定精度。在设计初期，利用理想公式确定拓扑和变压器匝比；在调试阶段，通过闭环反馈消除损耗和非线性因素带来的误差。

采样电路架构

  • 测量节点：电压采样电路的输入端连接至P点（车身地）。
  • 参考地选择：采样电路的参考地（GND）必须严格连接到电池包的总负极（）。这确保了ADC转换出的数字量准确代表公式所需的P点相对于电池负极的电压。
  • 硬件组成：信号采集通常由专用的模拟前端芯片，或由运算放大器、高精度电阻和ADC组合而成的调理电路实现。

信号分压与调理

  • 测量范围差异：P点的理论电压跨度为0V至，而主控MCU的ADC输入范围通常较小（如0至3.3V或0至5V）。
  • 分压网络：必须在P点与ADC输入引脚之间设计高阻抗分压电路，将宽范围的高压等比例缩小至ADC的安全采样范围内。
  • 软件还原：MCU读取ADC转换的原始数字量后，通过软件算法乘以硬件分压系数与转换系数，将其还原为真实的节点电压。

系统掉电电容支撑时间计算

系统掉电支撑时间的计算模型

在现代硬件系统中，负载不能简单等效为固定电阻，因此使用传统的充放电公式 来计算支撑时间是不准确的。根据负载在电压下降期间特性的不同，主要分为恒流和恒功率两种计算模型。

模型类型 物理特性与适用场景 计算公式 裕量影响评估
恒流模型 负载抽取的电流保持恒定 若误用于开关电源场景，会导致计算的支撑时间偏长，系统设计裕量不足
恒功率模型 现代系统通过 DCDC 开关电源供电的标准计算法。维持后级功率不变，随输入电压下降，抽取电流增大，电压跌落曲线呈非线性变陡 准确反映了现代高效率稳压转换器的真实工作状态，是工业界首选模型

工程选型建议
在实际电路图设计与物料选型中，不能直接使用理论计算值，必须预留 1.5 到 2 倍的设计裕量。这是为了应对铝电解电容的初始精度偏差（通常最差情况只有标称值的百分之八十）、长时间工作的老化缩水以及低温环境下的容量衰减。

  • 针对上述 24V 系统的实例，实际选型建议在输入端并联两颗 35V 或者三颗 35V 的铝电解电容。
  • 决定电容体积和成本的最大变数是最低截止电压 。如果后端选用更宽输入范围的 DCDC 模块（例如 9V 至 36V 均可工作），电容的能量就能被更深度地释放，所需的整体电容量将成倍减小。

系统掉电支撑电容的架构设计与布局

在进行系统级的掉电支撑设计时，储能电容必须放置在前端高压侧，通常位于 DCDC 转换器的输入端，严禁放置在其后端。该架构设计基于能量密度最大化以及宽压稳定运行两个核心工程原理。

能量密度最大化
电容储存的能量与电压的平方成正比。以 48V 转 12V 的系统为例，将电容放置在 48V 前端，仅需微小的电压跌落即可释放巨大能量。若放置在 12V 后端，由于电压仅为原来的四分之一，要储存同等能量，电容量必须剧增至十六倍，这将导致极高的硬件成本与巨大的物理体积，无法在工程上实现。

宽压稳定运行
将电容放置在前端可以充分利用 DCDC 转换器的宽输入稳压特性。掉电发生时，前端电容开始放电，电压从 48V 逐渐下降。只要电压未跌破 DCDC 的欠压锁定阈值，转换器就能持续稳定输出 12V 或 3.3V，后端敏感的数字芯片完全不受母线掉电波动的影响。如果将电容放置在后端，掉电瞬间电容直接给负载供电，电压会立刻从 12V 开始下降。部分数字芯片对电压极其敏感，极小的跌落即可导致系统复位死机，最终导致电容内剩余的绝大部分能量无法被有效利用。

电容分类 系统布局位置 容值量级 核心工程作用
储能支撑电容 前端高压侧（DCDC 输入端） 极大 利用高压储存能量并依托稳压器平滑释放，抵抗整体系统掉电
去耦或瞬态响应电容 后端低压侧（靠近芯片负载） 较小 在负载瞬间抽取大电流而环路尚未响应的微秒级时间内提供电荷补给，专用于抑制输出电压纹波
