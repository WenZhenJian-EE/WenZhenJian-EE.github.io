大功率系统安规设计的底层逻辑与避坑笔记

做高压变流器、大功率车载充电机（OBC）或者工业逆变器，安规（Safety）是绕不过去的一道坎。安规要是没设计好，测试时耐压打不过、击穿漏电，产品根本出不了厂。更别提这玩意如果在线上出故障，漏电把人给击了，那可是重大安全事故。

这篇笔记整理了我之前搞大功率电源时，在安规击穿整改、爬电距离设计上踩过的坑。顺便聊聊基于危险源的安全工程（HBSE）的防护思想，以及怎么在 PCB Layout 上少交点安规学费。

1. 能量源分级（ES1/ES2/ES3）与防护的本质

现在的安规标准（像最新的 IEC 62368-1）核心理念其实是基于危险源的安全工程（HBSE）。
HBSE 的防线思维：别去指望元器件永远不坏。做硬件的必须默认设备在生命周期内肯定会出故障。我们的设计任务是，当某个零件坏了、发生单点故障时，系统必须能以一种安全的方式挂掉，绝对不能让高压电传到用户能摸到的地方。

标准把电能量源分了三个等级：

  • ES1（安全能量）：人体摸到最多有点酥麻，不疼也不会受伤。直流不超过 60V，交流不超过 30Vrms。这就是我们常说的安全电压（SELV）。
  • ES2（临界能量）：摸上去会疼，但不会有生理损伤（比如以前的固话电话线电击）。这个级别需要加一层防护。
  • ES3（危险能量）：220V 市电、三相电、或者 800V 的高压直流母线全属于 ES3。摸到就可能致命。普通用户能碰到的部分和 ES3 之间，必须有双重绝缘或者加强绝缘的隔离阻挡。

2. 绝缘层级与 Class I / Class II 设备的防电击机制

安规里把绝缘分成了五个等级，但工程上我们重点关注这几个：

  • 功能绝缘（Functional）：仅仅为了让电路能工作（比如正负极之间防短路），不提供电击保护。
  • 基本绝缘（Basic）：第一道防电击防线。但安规假定它随时可能坏，所以它不能直接暴露给用户摸。
  • 附加绝缘（Supplementary）：在基本绝缘之外套的一层独立防线。
  • 双重绝缘（Double）：基本绝缘 + 附加绝缘。
  • 加强绝缘（Reinforced）：单层结构，但抗击穿能力等于双重绝缘（比如高频变压器的三层绝缘线，或者磁耦数字隔离芯片）。

Class I 设备的“PE 保命法”

我们在实验室搞的几十 kW 充电模块、工业逆变器，绝大多数都是 Class I 设备：

  • 做法：带高压的部件（ES3）和金属外壳之间，只要做基本绝缘就行。但有一个硬性规定：金属外壳必须接保护地（PE）。
  • 原理解析：一旦基本绝缘老化击穿，高压漏到了金属外壳上。因为外壳接了地线，高压电瞬间通过外壳、PE 线跑回大地（电源侧中性点），相当于造了一个“金属性短路”。几十上百安培的短路电流瞬间流过，瞬间触发配电箱的空气开关跳闸，把电断掉。
  • 重点：这个保护机制能起作用，全靠外壳接地的阻抗足够低。如果外壳接 PE 接得磨磨唧唧，接触阻抗太大，跳闸不及时，人去摸外壳还是会被击。所以 PE 的接地电阻测试必须要严格通过。

3. PCB 爬电距离（Creepage）与电气间隙（Clearance）怎么卡？

在 PCB Layout 时，安规间距必须在两个维度上卡住：

  • 电气间隙（Clearance）：两个导体在空气中的最短直线距离。它防的是雷击浪涌这种瞬态高压引起的空气电弧击穿。
  • 爬电距离（Creepage）：两个导体沿着板材表面的最短距离。它防的是长期直流高压下，板子表面的湿气和灰尘导电，导致 FR4 介质慢慢碳化成一条微小的漏电通道（漏电起痕）。

3.1 查安规表时的三个致命参数

画板前，必须弄清楚这三个关键参数，不然辛辛苦苦画出来的板子送检必挂：

  1. 过电压类别（OVC）：设备接在电网的哪个位置。直接连电网三相电（OVC III）要求的间距，会比插适配器的低压二次侧（OVC II）大一倍。
  2. 相比漏电起痕指数（CTI）：板材抵抗漏电起痕的能力。普通的 FR4 板子，CTI 通常在 （安规里的 IIIa 组别），抗漏电能力一般。
  3. 污染等级（Pollution Degree）：工作环境脏不脏。室外的充电桩按 3 级（有导电粉尘）计算，普通的室内机箱按 2 级计算。

3.2 800V 高压母线怎么缩减爬电间距？

如果做 800V 直流系统，用普通 FR4（CTI < 400），在 2 级污染下：

  • 硬性指标：基本绝缘要 间距，加强绝缘直接要 ！要是原封不动把间距留够 16mm，隔离采样板或者隔离驱动板的尺寸就大上天了，根本没法做小。
  • 工程上的“逃生通道”：
    1. 选高 CTI 板材：改用高 CTI 的车规级板材（CTI ，Group I 组别）。爬电距离要求立刻从 砍掉一半，缩到 。
    2. 物理开槽（Slotting）：在隔离两侧的焊盘中间，开一个宽度大于 的铣空槽。开槽把板子表面的湿气和微尘通道强行截断，爬电路径没法从板子表面过去，只能绕过空气槽的边缘。这样计算爬电距离时，所需的焊盘物理间距就可以极大地压缩。
