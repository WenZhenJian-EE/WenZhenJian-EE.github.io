  • 电路基础底层原理
  • 源的本质定义理解
  • 数值意义与波形数学计算

电压与电流：最基础也最容易被忽视的直觉

  1. 电压是相对的，地是人工捏的：电压只是两点之间的电势差，真实世界里没有绝对的零电位。我们在分析单板或者多板系统时，常说的“这点的电压是多少”，其实全是指它和系统参考地（GND）之间的电势差。如果地线上有高频干扰，你测到的所有信号都会失真。
  2. 电流必须要回娘家（闭环）：电流绝对不能凭空消失，它从系统正极出发，流过负载，最后必须顺着回路回到负极。电流在回流路径（哪怕是地线）上走过，必然会产生压降。如果你看到电流从 A 点流出却回不到 A 点，那不是你测错了，就是违反了能量守恒定律，系统正在“漏电”或者通过寄生电容向外幅射。
  3. 电流挑剔但也不死板（动态阻抗分流）：大家都听过“电流会挑阻抗最小的路走”。但在并联支路里，分流是按动态阻抗比例分的，并不是说高阻抗支路就完全不走电流。比如二极管反向截止时，它的阻抗极大，但依然会有微安级的漏电流。在敏感的采样电路里，这个小小的漏电流就能让你精度失准。
  4. 电流怎么逆天改命（从低电位流向高电位）：水往低处流，电流一般从高电势流向低电势。如果我们要让电流“逆天改命”往高处走，就必须借助于能储能的元件（电感）。电感把磁场能放出来强行推着电流走，在这个过程中，正向电流会慢慢变小直到零，然后反向爬升。
  5. 负压是怎么被吸出来的（ZVS与米勒效应的物理图景）：我们可以把电流流动等效为电荷的运送。如果电感强行把电荷从某个节点抽走，而别的地方来不及补充，这个节点就会形成“负压”（电荷稀缺）。这就解释了 ZVS 过程中，电容电荷被抽干后电压降到 0 甚至变负的现象。我们在调米勒电容时，也是利用这个“负压吸走电流”的思路来分析管子为什么会被瞬间关断。
  6. 强行钳位：是指用二极管或稳压管等硬件手段，把某个电路节点的电位死死按在特定值上，不给它乱晃的机会。

直流与交流：用“水力学”做个直观类比

  • 直流 (DC)：就像两个水位不同的水库，高水位水库（比如 12V）持续稳定地向低水位水库（比如 5V）放水。两个水库的水位差就是电压，水流就是电流，出水口的大小就是负载阻抗。阻抗越小，出水口越大，流过去的电流就越大。
  • 交流 (AC)：就像两个水库的水位在中心位置周期性地上下波动。频率是波动的快慢，幅值是浪高，相位是两个水库波浪起伏的时间差。
    • 如果两边波动的幅度、频率、相位完全一样，那水流在通道里只是来回晃，没有净水量交换（没有功率传输）。
    • 如果 PCS（储能变流器）侧的水位波动幅度比电网高，但相位完全一致，那水流在一个周期里刚好有来有回，交换的全部是无功功率，没有净输出。
    • 如果 PCS 侧的相位超前于电网，它会先于电网推高水位，水流向电网走（给通道里的电感储能）。随后即使 PCS 水位落后了，由于电感的惯性（水流惯性），水还会继续往电网流。这样在一个周期内，就实现了有功功率的净输出。

核心元件特性的研发直觉

1. 电感：桀骜不驯的“水轮机”

  • 物理本质：通过建立磁场储能，生来就是为了反抗电流变化的。
  • 电流不能突变：你如果强行去断开它，它会用极高的感应电动势把你的 MOSFET 烧毁。
  • 惯性水轮机类比：电感就像一个沉重的带飞轮的水轮机。你必须在两端建立水位差（电压）才能慢慢把它转起来（建立电流）。一旦它转起来了，由于巨大的转动惯性，你用手去强行按停它（开关断开），它会爆发出极大的反作用力把你的手震断（产生高压）。因此它对高频交流呈现高阻抗。
  • 磁饱和红线：电感磁芯存磁能是有极限的。电流太大了磁畴全部排齐，电感直接退化成纯铜导线，感量清零。

2. 电容：带弹簧的“储水气缸”

  • 物理本质：通过建立电场储能，死守两端电压不能突变。
  • 带弹簧的气缸类比：往气缸里注水（灌电流）会推动活塞压缩弹簧，活塞的位置就代表电容电压。对于直流电，当弹簧反弹力和水压抵消时，水流停止，等效开路。

3. 寄生参数：高频下的隐形魔鬼

任何实际器件在 PCB 上都不能看作理想元件：

  • 电容的真实模型：一个理想电容串联一个等效电阻（ESR）和等效电感（ESL）。在高频下，电容会过早地发生谐振，过了谐振点后它在电路里其实是个电感！
  • 电阻、电感的真实模型：一个理想阻感串联，再并联一个绕组寄生电容 C。

电路分析模型与视角

1. 谁是因，谁是果？

  • 电压是因，电流是果：最常用的逻辑。开关管动作，SW 点电压在 0V 和输入电压之间剧烈跳变（因），这个高 的电压跳变驱动电流在电感和地线之间流动（果），从而产生共模噪声。
  • 电流是因，电压是果：对偶逻辑。在分析光伏面板、工作在放大区的三极管，或者开关管断开瞬间的电感续流时，我们更倾向于把它们看作电流源。电感强行输出电流，遇到阻抗后在节点处激发出电压。

2. 戴维南与诺顿等效：殊途同归

  • 戴维南等效：把线性网络等效为“电压源 + 串联电阻”，用来分析“电压是因”的电路。
  • 诺顿等效：把线性网络等效为“电流源 + 并联电阻”，用来分析“电流是因”的电路。
  • 我们的工程分层思维：
    • 粗估分析：如果源内阻比负载阻抗小得多，直接看作理想电压源；如果源内阻比负载大得多，直接看作理想电流源。
    • 细调分析：一旦涉及电压波动、瞬态跌落、噪声分析，必须把“理想源 + 内阻”的物理模型带进去算。

3. 稳态阻抗与暂态响应：别拿频域公式算上电瞬间

  • 交流稳态（）：这套复数代数运算是基于正弦波稳定振荡的假设。它能帮你快速算出交流滤波器的截止频率，但它绝对不适用于分析上电合闸、开关管切相瞬间的暂态过程。
  • 暂态分析：必须回到时域，分析上电瞬间（）电容等效短路、电感等效开路的物理边界。
  • 非正弦波（方波/三角波）怎么算：别直接把频率代入 算阻抗。我们要用傅里叶级数把它拆成基波和各次谐波，对每个正弦谐波分量单独求阻抗响应，最后用叠加定理把它们加起来。这才是我们在做电容滤波设计时的真实数学图景。

研发分析的黄金守则

  1. 电流一定要闭环：分析任何拓扑，先用手指在图纸上画出电流的去路和回流路径，确保状态受控、路径清晰。
  2. 忽略标签看本质：看复杂的系统图时，先把那些“+5V”、“+12V”的网络标签在脑子里抹掉，回归到基本的 KCL、KVL 以及二极管、电阻的物理导通特性上。很多多板级联系统发生的“地回流交叉干扰”，就是这样抓出来的。
  3. 以平衡为起点，用扰动看动态：分析反馈系统时，先假设它工作在完美的设计平衡点。然后，给系统输入一个微小的电压扰动或者负载跃变，看它在时域里是怎么一步步过渡、收敛到新平衡点的。

经典电路结构解析

1. 桥臂：拓扑的能量闸门

桥臂是电力电子的核心单元，本质上就是一个受控的单刀双掷开关。桥臂的中点作为能量的输入或输出节点。

2. 电感储能式变换器：CCM 的调控本质

在连续导通模式（CCM）下，Buck、Boost 的输出电压完全由占空比 D 决定。如果在一个周期里，充电伏秒积比放电大，电感电流的直流底座就会像爬楼梯一样逐周期往上爬。

  • 恒压模式的调整过程：负载突然变重（拉电流增加），为了维持输出电压，控制环路会在几个周期内把占空比拉大一点（比如从 30% 短暂调到 32%）。在这段动态调整期，电感电流底座快速爬升，多余的交流纹波通过输出滤波电容吸收，从而保证输出电压平稳。当电感电流平均值赶上负载电流后，占空比又会退回 30% 的稳态值。
  • 恒流模式的调整过程：环路死守电感平均电流。当占空比发生变动时，输出电压会被动随之漂移，在这个过程中，电感电流的纹波大小会发生剧烈波动。

3. 开关引脚的隐藏状态：高阻态避坑

  • 引脚不等于 0 和 1：我们在调主控或者驱动芯片的 PWM 引脚时，当它输出“关闭”时，它不一定会把引脚死死接地（输出低电平“0”），很多芯片会把它设为高阻悬空状态（Hi-Z）。这时候如果不加下拉电阻，引脚电平会被 PCB 上的杂散噪声强行带起来，导致管子莫名其妙地导通直通。
  • 电源等效大电容：在系统高频分析中，所有的直流电压源在交流阻抗上都可以等效为一个并联的大电容，因为它们在输出端都挂着厚重的滤波电容。

2. 功率因数 (PF) 怎么估？

功率因数 。降低 PF 的主要有两个凶手：一是电压电流相位差（位移功率因数 ），二是负载非线性导致的电流波形畸变（畸变功率因数）。

我们在现场估算设备功率因数时，常按负载类型分类：

  • 纯阻性负载 (PF ≈ 1.0)：电热管、白炽灯。有功等于视在。
  • 感性负载 (PF ≈ 0.5 - 0.9)：电机、变压器。有功远小于视在，需要并联电容柜做无功补偿。
  • 现代开关电源：如果带了主动 PFC（功率因数校正）电路，PF 可达 0.95 以上；如果是廉价的不带 PFC 的小充电器，电流呈断续尖峰脉冲，PF 可能低至 0.5。
