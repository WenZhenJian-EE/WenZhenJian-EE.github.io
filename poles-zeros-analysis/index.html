开关电源控制建模与 Bode 图调试避坑笔记

搞开关电源的，控制建模和零极点分析绝对是个硬骨头。很多人喜欢在电脑上用 Simplis 或者 LTspice 仿真，跑出一个漂亮得不真实的波形，结果一去实验室，用网络分析仪（比如 Bode 100）实测，扫频曲线满是杂波，甚至直接震荡报警。

这篇笔记整理了我调大功率变换器时的环路控制设计和扫频实战经验。咱们抛开枯燥的控制理论公式，直奔物理本质，聊聊右半平面零点（RHPZ）的致命之处、电压/电流模式的简化，以及实验室扫频测试怎么避坑。

1. 零极点的直观物理理解

在传递函数里，零点和极点可不只是复数解，在电路里它们是有对应的物理直觉的：

  • 极点（Pole）：系统的阻碍者（惯性环节）
    • 物理概念：极点代表电容充电或者电感电流建立的惯性。时域上表现为阻碍变化，频域上相当于低通滤波器。当输入信号频率太高时，系统因为惯性跟不上了，输出幅度开始衰减，并且相位产生滞后。
    • 波特图表现：每遇到一个左半平面极点，幅频增益斜率下降 ，相位滞后 。
  • 零点（Zero）：系统的超前者（预测/微分环节）
    • 物理概念：零点代表微分响应（像电容两端电压的变化率）。它在频域表现为高通滤波器，能预测信号的变化趋势，提供相位超前，提升系统的动态响应速度。
    • 波特图表现：每遇到一个左半平面零点，幅频增益斜率上升 ，相位超前 。

2. 右半平面零点（RHPZ）：大功率拓扑的稳定性噩梦

在 CCM（连续导通模式）下工作的 Boost、Buck-Boost 和 Flyback 拓扑中，传递函数里必然会出现右半平面零点（RHPZ）。这玩意是控制环路设计中最让人头疼的存在。

2.1 时域上的“逆效应”

以 CCM 的 Boost 为例。当输出负载突然增大、电压掉下来时，环路控制会命令占空比 瞬间增大，想让开关管导通时间变长以注入更多能量。
但物理过程是这样的：

在占空比增大的这几个瞬态周期里，因为开关管开通时间变长，二极管导通时间（1-D）反而被压缩了。输出电容得不到能量补充，导致输出电压在短时间内进一步下跌！这与我们想要提升能量的初衷完全相反。
等几个周期后，电感电流慢慢爬上来，输出电压才会回升。

这种“占空比增大，电压反而先往下跌”的逆向响应，在频域上就是右半平面零点。

2.2 频域上的矛盾与阻碍

  • 数学形式：。
  • 波特图特征：它的幅频特性和普通零点一样，使增益曲线斜率上升 ；但它的相频特性却和极点一样，引入了 的相位滞后！
  • 设计大忌：千万别想着放一个极点去对冲 RHPZ。对冲不仅不会挽救滞后的相位（相位会狂跌 ），还会把高频增益拉得极高，高频噪声被无限放大，环路必死无疑。
  • 唯一的工程解法：认怂。强行把控制环路的带宽（穿越频率 ）压低在 RHPZ 频率的 以下。控制环路反应慢一点，系统就会自动“忽略”刚开始的那个逆向压降瞬态，等电感电流建立好了再进行调节。

3. 电压模式（VMC）与电流模式（CMC）的被控对象建模

被控对象的建模阶数直接决定了我们调试补偿环路的难易程度。

3.1 电压模式控制（VMC）：二阶 LC 系统

  • 本质：直接控制占空比 。电感和电容构成一个二阶低通滤波器。
  • 极零点：在谐振点 处存在共轭双极点。
  • 调试痛点：跨过谐振点时，增益以 断崖下跌，相位突降 。如果不放一个 Type III 补偿器去放置双零点对冲，系统极易振荡炸管。

3.2 电流模式控制（CMC）：降阶为一阶 RC 系统

  • 本质：引入电流内环。因为电流内环极快，电感在低频段等效为一个受控电流源，不再作为惯性元件参与能量交换。
  • 极零点：电感极点消失，系统降为一阶系统！只存在一个由负载和输出电容决定的单极点 。
  • 优势：相位在低频段最大只滞后 ，相位裕量非常充沛。只要用一个简单的 Type II 补偿器（单零点、双极点）就能搞定高带宽设计。

4. 实验室扫频测试（Bode 100）避坑法则

在实验室测 Bode 图时，如果不注意接线和信号阻抗，扫出来的曲线会惨不忍睹。

4.1 注入电阻切在哪里？

必须选择反馈回路中高阻抗输出端指向低阻抗输入端的节点切断，串入一颗 的注入电阻（通常是在反馈分压电阻的高端脚上）。这样才能确保注入的交流 AC 扰动单向传输，不破坏原有静态工作点。

4.2 注入信号幅值的微调妥协

  • 信号太大（自激饱和）：低频段环路增益很高（比如 60dB），如果注入信号太大（如 1V），会使误差放大器或者比较器进入饱和限幅状态，扫出来的增益曲线变平，相位失真。
  • 信号太小（噪声淹没）：高频段环路阻抗和增益衰减严重，如果注入信号只有 10mV，会被开关电源的高频开关噪声完全淹没，高频段曲线全是一堆锯齿毛刺。
  • 正确姿势：采用动态变幅值扫频。低频段用小信号（如 ），高频段逐步拉大信号（如 ），保证全频段的信噪比。

4.3 隔离变压器（注入变压器）必不可少

扫频仪的输出端口地和主板地绝不能直接硬连，否则会引入地环路噪声，甚至造成短路烧毁设备。必须使用高频隔离注入变压器（如 B-WIT 100），进行交流耦合，确保安全和波形纯净。
