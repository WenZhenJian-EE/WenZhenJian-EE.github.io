双向与交错并联 Buck-Boost 变换器设计与调试手记

平时做电源开发，Buck-Boost 绝对是接触最多的拓扑之一。从最基础的单向 Buck/Boost，到双向充放电、交错并联以减小体积，再到高压下的三电平拓扑，里面有太多只有在实验室里用示波器抓波形、焊铜箔才能体会到的“暗坑”。

这篇长文是我这几年在做双向 DCDC 变换器和交错并联拓扑时的设计、仿真与调试笔记。内容包括控制建模、MOS 管背靠背保护、PLECS 仿真避坑、环流问题以及三电平电容均压控制，防忘，也希望能帮到同行。

一、 双向 Buck-Boost 的“势”与电流控制

双向 Buck-Boost 的能量流动方向并不取决于 PWM 信号本身，而是两侧电压的“势”与 PWM 占空比共同作用的结果。其底层逻辑是电感的伏秒平衡原理。

如果在我的样机里，高压侧为 48V，低压侧为 24V，那么平衡点占空比 。实际电流的流向取决于你的占空比 和这个平衡点的关系：

  • D > 0.5 (例如 D = 0.6)：中点平均电压 。因为 ，电感上产生正向压差，电流从高压侧流向低压侧，即工作在 Buck 模式。
  • D < 0.5 (例如 D = 0.4)：中点平均电压 。因为 ，电感上产生负向压差，电流从低压侧流向高压侧，即工作在 Boost 模式。
  • D = 0.5：中点平均电压刚好是 24V，两侧电位相等，没有功率传输，属于待机状态。

这就像两个水箱，D 控制着中点水箱的虚拟水位。水位高于低压侧就向低压侧注水，低于低压侧就从低压侧抽水。

电流模式控制的必要性

当双向 DCDC 连接在两个固定电压源（比如电池和母线）之间时，它的本质是一个“可控电流源”。你千万不能直接去控制电压，因为两端电压已经被源强行钳位了，强行控压只会让环路直接自激打架。

在实际的硬件设计中（如使用 TL494 或 DSP），我们需要设计正、负双向电流环：

  • 正电流环（Buck）：电流增大则采样电压升高。当电流超过设定阈值时，拉高控制引脚，迫使占空比减小以限制电流。
  • 负电流环（Boost）：电流越负（即反向放电越大），采样电压越低。当低于反向保护阈值时，限制占空比以防过放。
  • 死区休眠段：在正负限流点之间，两个误差放大器均不动作，系统处于正常调节段。

安全警告：纯电流环控制极度危险。如果在空载或电池充满（阻抗极高）时，系统为了强行达到设定的参考电流，会无限制地开大占空比，导致输出电压失控飙升炸管。因此，必须引入电压环进行过压（OVP）兜底保护，可以通过 TL431 配合二极管在超压时强行拉低驱动，或者使用主控芯片的 DTC（死区控制）引脚限制最大占空比。

二、 假负载测试的“翻车”隐患与调试方案

在实验室测试双向 DCDC 时，如果我们高低压两侧都用普通的直流可调电源，会遇到一个大坑：单向可调电源只能输出电流，无法吸收电流（灌入电流）。

如果你在测试 Buck 模式时，DCDC 往低压侧的 5V 电源里灌电流，因为 5V 电源无法吸纳这部分能量，电荷会堆积在电源的输出电容上，导致这个节点的电压瞬间失控飙升，轻则电源触发 OVP 保护关机，重则直接烧毁 DCDC 板子。

假负载分流方案

为了在没有双向电源或电子负载的条件下完成调试，我们必须在节点处并联一个阻值合适、功率足够的假负载电阻：

  • 原理解析：在 5V 节点并联一个 的电阻，电阻会恒定消耗 1A 电流。当 DCDC 未工作时，5V 电源输出 1A 维持电压。当 DCDC 启动并往该节点灌入 0.4A 电流时，5V 电源内部的误差放大器检测到电压有上升趋势，会自动“退让”，把自己的输出电流削减到 0.6A。这样，电阻需要的 1A 电流由电源（0.6A）和 DCDC（0.4A）共同提供，节点电压依然稳在 5V。
  • 致命盲区：这个方案的边界是DCDC 输出电流不能超过假负载的消耗极限。如果 DCDC 强行输出 1.2A，而假负载只能消耗 1A，电源即使退让到 0A 也是退无可退。多余的 0.2A 只能继续灌入假负载，节点电压会被实打实地抬升到 。如果此时后级器件耐压不足，就会瞬间烧毁。所以，假负载的阻抗一定要留足裕量。

三、 控制环路设计与 PWM-ADC 同步采样避坑

双向半桥拓扑在硬件驱动上，高侧管 S1 和低侧管 S2 必须保持互补导通（带死区），使系统运行在强制连续导通模式（FCCM）。在这个模式下，即使电流降到零也不会断续，而是会反向流过 MOSFET 的沟道，从而保证控制的连续性。

电流采样与同步触发大坑

电流采样通常有线上采样和低边采样两种。

  • 线上采样：直接在电感和电容之间串分流器。波形连续，但需要昂贵的高共模抑制比差分放大器。
  • 低边采样：在下管源极与功率地之间放采样电阻。共模电压几乎为零，成本低，但是采样到的电流波形是不连续的梯形脉冲。

对于低边采样，如果你把 ADC 采样时刻放任不管，测出来的数全是高频噪声和无意义的零电平。

  • 黄金法则：在中央对齐（向上/向下计数）模式下，必须把 ADC 的硬件触发点死死锚定在下管导通区间的正中间。
  • 原因分析：下管导通区间中点时刻的瞬时电流，在数学上精确等于这一个周期的电感平均电流！而且这个时刻距离开关管切换点（高 噪声源）最远，开关引起的电压振铃已经衰减干净，电磁环境最安静，采样精度最高。
  • 根据驱动极性配置，这个中点要么对应计数器的 0 值，要么对应最大值 MAX。在软件中一定要做对应的寄存器映射（如 ），保证占空比控制方向一致。

四、 双向 Buck-Boost 输出端的“背靠背”隔离防线

在双向 Buck-Boost 拓扑（如电池充放电通道）的输出端，通常会设计两个背靠背（Back-to-Back）串联的 N 沟道 MOSFET 作为保护开关。

为什么必须用两个管子？

因为 MOSFET 内部存在寄生体二极管。如果只用单管，即使你关断了 MOSFET 沟道，电流依然可以顺着体二极管单向导通，无法实现真正的双向断开。

  • 共漏极（Common Drain）接法：在低端开关设计里，我们常把两个管子的漏极连在一起。由于 的源极直接接系统地（GNDO），主控芯片可以直接输出相对于地的驱动信号控制 。一旦 导通，公共漏极被拉低，再去驱动 就会变得非常简单。
  • 核心保护功能：
    1. 电池防反接：当外部电池接反时，若没有 的截止作用，电流会顺着半桥下管的体二极管和 的体二极管形成直接的短路回路。背靠背结构能完美截止这个反向电流。
    2. 过流/短路保护：通过 20mΩ 的检流电阻实时监控输出。一旦发生短路，瞬间拉低 和 的栅极，物理切断整个回路。
    3. 端口残余电量隔离：关机后，背靠背开关彻底切断输出电容与外部端口的物理连接，防止输出端火花放电。

五、 交错并联 Buck 仿真与磁耦合实战

理解交错并联拓扑，最快的方法就是在 PLECS 里搭个两相交错并联 Buck 模型。两路驱动信号需在相位上相差 180 度。

1. 独立电感与耦合电感（磁集成）对比

在标准交错并联中，我们用两颗独立的电感。如果想要进一步压缩体积、提高效率，可以用耦合电感（Mutual Inductor）进行磁集成。

  • 反向耦合铁律：在 PLECS 中连接 Mutual Inductor 时，必须让两相电流从相反的极性流过绕组（一相接同名端流入，另一相接同名端流出）。这样，两相直流电流产生的静态磁通方向相反，能在磁芯内部相互抵消，防止磁芯饱和。
  • 避坑：如果不小心连成了同向耦合，两相磁通叠加，等效电感会暴跌至 （自感减互感），导致电流纹波瞬间失控飙升，仿真直接发散报错。

2. 仿真自激与软启动设计

在 PLECS 中，如果突然给一个固定的占空比（阶跃信号），由于输出 LC 滤波器阻尼极低，电压和电流在启动瞬间会产生非常剧烈的低频共振波（固有谐振频率 ）。

为了解决这个谐振冲击，控制上必须使用载波比较法实现软启动：

利用斜坡信号（Ramp）与两个在相位上错开 180 度的三角波载波进行比较，让占空比从 0 缓慢爬升到设定值，这样就能平滑地建立输出电压，避免开机电流冲击。

六、 高压三电平交错并联的均压与采样

在 800V 甚至更高压的直流系统（如光伏逆变、车载充电机）中，三电平交错并联 Buck 是标配。它能将开关管的电压应力限制在母线电压的一半（），从而允许选用更低耐压、超低内阻的 MOSFET。

1. 控制参考地的抉择与母线电压采样

三电平电路中有上管电容 和下管电容 ，控制地（GND）的接法直接决定了采样的复杂度：

  • 接负母线：系统最低电位为参考地。能直接测出下管电容电压 和总电压 ，上电容电压靠软件相减（）得出。大部分非隔离降压系统采用此方案。
  • 接中点：以电容中点为参考地。此时正母线为正压，负母线为负压。控制芯片的 ADC 无法直接测负压，必须用运放搭建反相电路或抬升网络，硬件略显复杂。
  • 隔离采样（工业标配）：控制地与高压地完全隔离。利用隔离运放（如 AMC1301）配合光耦/磁耦将信号传回 MCU，抗瞬态共模干扰（dI/dt）能力极强。

2. 滤波电容强接中点的致命风险

在单相直流 Buck 系统里，输出滤波电容绝对不能强行接在母线电容的中点上。

  • 机理解析：如果是交流逆变器，流入中点的正负半周电流在长时间跨度下均值为零，中点电位能自平衡。但如果是直流降压，负载的直流回流是直接走负母线的，不经过中点。如果电容强接中点，会导致电容被单向持续充放电，中点电压瞬间严重偏置，上下电容电压失衡，开关管直接过压爆炸。
  • 直流系统的电容接法：输出滤波电容必须接负母线。
  • 对称型交错并联的救赎：在对称型双半桥结构中，上半桥工作时对下电容充电，下半桥工作时对下电容放电。配合通常相差 180 度的交错并联控制，注入与抽出的直流净电流在物理上天然抵消。但即便如此，软件中依然必须加入主动均压监控环路，当监测到两路电容电压偏离时，微调上下两相的占空比以实施强制电荷拉回，确保高压系统绝对安全。
