从实验室炸管到数模干扰：PCB Layout 避坑手记

画 PCB 可不是简单的“拉线工”，光指望 DRC 不报错就想一次投板成功是不可能的。尤其是在大功率开关电源或者高精密模拟系统里，Layout 画得不好，轻则控制环路自激、ADC 采样满是毛刺，重则开机直接击穿开关管。

前阵子调一块 SiC MOSFET 板子，因为一个过冲电压问题在实验室焊了三天。这里结合排查经历，总结几个 Layout 上的硬核避坑法则，主要聊聊高频大电流环路（Hot Loop）、驱动走线、数模混合接地以及散热孔的工艺大坑。

1. 压倒一切的铁律：最小化 Hot Loop（高频大电流环路）面积

在 Buck、Boost 这类开关电源里，管子一开一关，会产生极高 的变化电流。这个电流突变的路径就是 Hot Loop。

痛痛的教训：之前画的一块 Buck，输入去耦陶瓷电容 C_in 距离开关管 Q1 隔了大概 20mm。投板回来测试，关断瞬间漏源极（D-S）直接激荡出了一个 200V 的过冲电压尖峰。原本管子耐压也就 600V，母线 400V 加上这个 200V 过冲，直接把管子打穿了。

  • 怎么做：输入去耦陶瓷电容 必须无条件物理上贴死开关管的输入引脚。保证高频开关瞬态电流完全在 C_in 到管子的这几个毫米级环路内流动，将回路寄生电感压低到最小。
  • SW 节点铜皮别铺太大：开关管中点走线（SW 节点）是高 的电场辐射源。在满足载流的前提下，这块铜皮面积越小越好，否则会通过空间电容把干扰直接耦合到旁边的反馈线和采样线上。

2. 栅极驱动走线：像高频传输线一样去对待

驱动回路（驱动芯片 OUT -> 栅极 Rg -> 开关管 Gate -> 开关管 Source -> 驱动芯片地）看似普通，其实是个高频脉冲网络。

  • LC 谐振振铃：如果驱动走线太长、环路太大，寄生电感和管子的输入电容 就会发生谐振，在 Gate 极上激荡出严重的振铃。关断时的振铃要是高过了门槛电压，管子就会误开通，直接上下管直通炸机。
  • 避坑指南：
    1. 驱动芯片的旁路去耦电容（ 陶瓷电容）必须紧贴芯片的电源和地脚。
    2. 驱动线（Gate）和地返回线（Source）要紧密平行走线，或者在上下相邻层重叠走，最大化磁场抵消来降低电感。
    3. 驱动回路能不打过孔就别打。一个过孔差不多就是 1nH 的电感，多打两个过孔，加速关断的效果就全泄了。
  • 四脚管的 Kelvin Source 接法：现在很多大功率开关管（比如 TO-247-4L 封装）把源极分成了功率源极（Power Source）和信号源极（Kelvin Source）。驱动的地线必须单独单点接在 Kelvin Source 引脚上，绝对不能在 PCB 表面直接和主功率大电流的源极铜箔混接。这样可以把大电流引起的源极压降和驱动回路彻底隔开，防止地电位被主功率污染。

3. 数模混合接地：别再盲目割地了

以前很多低频设计手册里教大家把模拟地（AGND）和数字地（DGND）彻底割开，最后通过一个 0 欧电阻单点连接。但这在高速、高集成的板子上往往会起反作用。

高频信号的回流电流永远选择**阻抗最小（也就是包围环路面积最小）**的路径，通常就在信号走线正下方的参考地层。
如果你在地层上划了一道缝去隔开数模地，而信号线又不得不跨过这道缝：

这时候回流电流无法从裂缝穿过去，只能绕一大圈找通路。结果导致信号环路面积暴增、阻抗飙升，波形严重畸变，而且这条缝隙等效于一根缝隙天线（Slot Antenna），会疯狂往外辐射高频噪声，EMI 测试直接报废。

更科学的接地哲学：完整地平面 + 空间分区

  1. 地平面保持完整不割裂，提供阻抗最低的电流回流通道。
  2. 严格按功能分区：把敏感的模拟电路（如分压采样、运放）和高频开关的数字电路（如 MCU、时钟、DCDC 控制）物理隔离在板子的不同区域。
  3. 高频电流天然选择阻抗最低的路径，数字部分的噪声电流会自动局限在数字走线下方的地层里，模拟部分的电流局限在模拟走线下方，只要不产生空间交叉，两者在完整地层上互不干扰。
  4. 为了保险，可以在运放的模拟电源输入端加上电容-磁珠-电容（CLC）的 π 型滤波器，阻断高频噪声通过电源线串扰。

4. 散热过孔（Thermal Vias）与焊盘打孔工艺

大功率芯片或者 DPAK/QFN 封装的管子下方，通常需要打散热孔把热量传到背面铜皮。

  • 为什么要这个尺寸：过孔小于 0.2mm，PCB 厂挂铜厚度不够，热阻太大，散热效果打折扣；如果大于 0.4mm，回流焊时液态焊锡会在毛细效应下顺着过孔漏到板子背面（俗称“偷锡”），导致正面器件脱焊虚焊。
  • Via-in-Pad（焊盘藏孔）必须塞树脂：如果管子封装特别小，散热孔只能打在焊盘内部（Via-in-Pad），在发给 PCB 厂加工时必须指定“树脂塞孔 + 电镀填平（POVP）”工艺。常规的油墨塞孔根本做不平，回流焊时会漏锡或者导致管子偏斜。必须用高导热树脂塞满，表面再镀铜拉平，才能确保焊接贴合度。
