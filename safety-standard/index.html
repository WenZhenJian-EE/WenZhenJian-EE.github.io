安规标准收录、解读

我们在研发中经常要和各种安规、EMC、可靠性标准打交道。很多兄弟一听到“标准”两个字就头大，觉得都是些枯燥晦涩的条文。其实，标准就是前人踩了无数个雷、炸了无数个管子之后总结出来的“排雷指南”。今天我结合手头做过的一些高压大功率产品，把我们在研发中天天翻阅的核心标准，以及怎么去“读懂”那些复杂的安规表格，用我们搞硬件的大白话做个彻底的梳理。

我们天天在查的常用标准

1. 通用 PCB 设计与绝缘配合标准

  • IPC-2221B (通用 PCB 设计标准)
    • 我的理解：这算是个“行业通识”。它从工艺制造的角度规定了 PCB 走线线宽、线距和耐压的底线值。
    • 避坑点：千万别直接用 IPC-2221B 的间距表格去设计安规隔离带！ 因为这个标准根本没有把污染等级、过电压类别这些关键的安规要素考虑进去。如果你只盯着 IPC-2221B 划定原副边间距，去过安规认证时保证被裁判直接拍死。它只能用来作为板子内部非安全电路（比如控制信号走线线距）的制造参考。
  • GB/T 16935 (等同于 IEC 60664-1, 低压系统内设备的绝缘配合)
    • 我的理解：这才是真正的安规底层母标准。它解释了为什么我们需要拉开距离，以及怎么根据工作电压、瞬态过电压、污染等级和板材的相对漏电痕迹指数（CTI）来计算具体的电气间隙和爬电距离。不管是做逆变器、充电桩还是家电，绝缘计算的源头都在这里。

2. 产品类安全标准

  • IEC 60950-1 与 IEC 62368-1 的演进
    • 旧的 60950-1 就像是一本硬性规定的教科书，直接给你条文和数字。而新一代的 62368-1 引入了 HBSE（基于危害的安全工程） 理念，它不强制说你必须留多少 mm，而是让你去识别你的产品里有哪些危险的“能量源”（比如 ES3 级高压），然后分析这个能量源会通过什么路径传导给人，接着要求你设计相应的物理屏障（防护）来隔离它。
  • PCB层间介质厚度设计推荐 (参考 IEC 62368-1)
    安规认证时，考官对 PCB 垂直层间（比如原边线和副边线在上下层重叠）的绝缘要求非常高。因为层压过程中一旦有杂质或空洞，高压一下就穿了。以下是我们在工艺设计时指定的最小介质厚度：
绝缘等级 典型应用场景 推荐最小介质厚度 我的工艺备忘
功能绝缘 (Functional) 同一电路内部 (如 5V 信号层与 3.3V 电源层之间) ~0.1 mm (4 mil) 通常单张最薄的 PP 片（半固化片）就能搞定。
基本绝缘 (Basic) 提供基础防电击保护 (如 110V 原边侧电源层与地层之间) ~0.2 mm (8 mil) 必须确保介质厚度，一般工厂建议至少叠两张薄 PP，防止单张针孔缺陷。
加强绝缘 (Reinforced) 隔离危险高压与安全低压 (如初级侧与次级侧在多层板中重叠) ≥ 0.4 mm (16 mil) 安规审查的重中之重！ 必须在压合结构图（Stack-up）里指定使用至少两张甚至三张 PP 片，绝对不能用单张 PP，以防单点电击穿。

电磁兼容性 (EMC/EMI) 常用标准

过安规只是保命，过 EMC 才是真功夫。我们的设备不仅不能变成一个大辐射源去干扰别人，还得能扛住别人的电磁骚扰。

1. 发射类 (Emission)

限制我们的设备往外排电磁“垃圾”：

  • CISPR 11 / EN 55011：工科医（ISM）设备的发射限值。
  • CISPR 32 / EN 55032：多媒体、IT 类设备的发射限值。
  • IEC 61000-3-2：谐波电流限制。搞主动 PFC 电路的兄弟对这个最熟悉，如果你的输入电流畸变率（THD）超了，或者低次谐波高了，过不了这关就别想在欧洲卖。
  • IEC 61000-3-3：电压波动和闪烁限值。

2. 抗扰度 (Immunity)

考核我们设备挨揍时的耐受力：

  • IEC 61000-4-2 (静电 ESD)：防的就是冬天干燥时人手去摸接口的那一下。
  • IEC 61000-4-3 (辐射抗扰 RS)：防外界无线电波对我们控制电路的干扰。
  • IEC 61000-4-4 (脉冲群 EFT)：测试电源线和信号线上由于开关断开等原因引入的快速瞬变尖峰脉冲。
  • IEC 61000-4-5 (雷击浪涌 Surge)：大功率设备（逆变器、大电源）的鬼门关。通常要打差模 2kV、共模 4kV 以上的高压冲击，压敏电阻（MOV）和气体放电管（GDT）选型不好，或者板上间距没留够，打浪涌时直接“啪”地拉弧放电，当场炸毁。
  • IEC 61000-4-6 (传导抗扰 CS)：射频场感应的传导干扰。
  • IEC 61000-4-11 (跌落与暂降)：网电压突然抖一下或者断个几十毫秒，看我们的电源会不会挂掉。

各行各业的特定安全与可靠性规范

做不同的项目，对应的顶层标准完全不同：

  • 测量控制仪器：走 IEC 61010-1。
  • 电机驱动变频器：看 IEC 61800-5-1。
  • 光伏逆变器/储能变流器：走 IEC 62109-1 / -2，这里对直流 1500V 系统的绝缘要求有专门的严苛条款。
  • 医疗电气：那就是大名鼎鼎的 IEC 60601-1。医疗安规对漏电流限制极其变态（微安级），隔离必须上双重隔离，患者接触端甚至需要两层加强绝缘。
  • 北美并网准入：UL 1741 绕不开。
  • 车规电子：功能安全走 ISO 26262，元器件可靠性看 AEC-Q 系列。
  • 外壳防护与试验：IEC 60529（IP 防护等级，防尘防水）；IEC 60068 系列（高温、高湿、振动等环境可靠性测试）。

实战中怎么套用这些标准？

我们在开发一个新项目时，可不是盲目去翻标准，而是有一套标准的闭环流程：

  1. 搞清卖给谁、在哪卖：确定是工科医、光伏还是家用，以及销售地区（国内 CCC，欧洲 CE，北美 UL）。
  2. 锁定顶层标准：比如设计一款户用光伏逆变器，先抱住 IEC 62109。
  3. 拆解硬指标：从顶层标准里抠出设计底线。比如输入 1000V，要求加强绝缘，EMC 要过 B 类限制，雷击浪涌要扛共模 4kV。
  4. 翻阅底层标准干活：
    • 画 PCB 时，具体留多少间距，调出 GB/T 16935 (IEC 60664) 查表。
    • 做 PFC 电路和输入 EMI 滤波器设计，按照 IEC 61000-3-2 计算谐波余量。
    • 进行浪涌摸底时，按照 IEC 61000-4-5 的测试波形在实验室里打压整改。

安规表格解读：别被那些天书一样的数字唬住

安规标准的附录里通常有一堆表，很多兄弟看得一头雾水。我总结了几个查表的底层逻辑和黑话：

1. 高海拔的空气大坑

安规表格里的间距都是以海拔 2000 米以下为基准的。如果客户要求设备必须能在西藏、青海等高海拔地区运行，电气间隙必须乖乖乘以修正系数。空气越稀薄，越容易被电击穿！

2. 污染等级的物理意义

爬电距离最怕的就是表面脏了漏电。污染等级反映了你产品的使用环境：

  • 污染等级 1: 最干净。比如完全灌了胶、或者涂了厚厚三防漆的 PCB 内部。这种地方基本不需要考虑漏电起痕，间距可以缩到极致。
  • 污染等级 2: 普通室内。常规办公室、服务器机房。虽然偶尔会因为温差结露，但平时还是干燥非导电的。我们大多数工业/消费类裸露 PCB 外层设计默认按污染等级 2 查表。
  • 污染等级 3: 工业现场。粉尘、湿气很重，或者露天没有防护的地方。爬电距离必须拉得很开。
  • 污染等级 4: 极端恶劣。比如经常被雨淋的露天设备，导电性粉尘漫天飞。

3. 怎么查电气间隙？(以 Table 2K / Table 2L 为例)

在旧的标准里：

  • 第一个电气间隙 C1 (Table 2K)：这是无条件的。直接看你系统能承受的最大脉冲耐压。比如系统直流母线 420VDC，网口浪涌要求耐压 2500V。在污染等级 I 或 II 类下，查 Table 2K 就能得出，功能绝缘（F）要求 1.5mm，基本绝缘（B）要求 2.0mm。
    • 解方程秘籍：如果你的工作电压刚好夹在表格里两个档次之间，安规表的值实际上是一个二元一次方程（线性关系），而不是直接等比例算。假设峰值电压 420V 对应基本绝缘（Base）是 2.0mm，840V 对应是 3.2mm，如果你的电压是中间的值，只要列个一次方程，解出中间值即可。
  • 第二个电气间隙 C2 (Table 2L)：这取决于你的系统电压超过电网瞬态电压的程度。最终的电气间隙需要 C1 + C2。

4. 怎么查爬电距离？

  • 只看交流的有效值 (RMS) 或直流的额定值。
  • 确认你所用 PCB 板材的 CTI 组别（一般普通 FR-4 板材默认归为 IIIb 或 IIIa 组，CTI 比较低，在 175V 左右）。
  • 结合污染等级和有效值电压去对应表格查表，得出具体的爬电距离。如果板上空间不够，要么换高 CTI 板材（比如 I 组，CTI 600V），要么开隔离槽。
