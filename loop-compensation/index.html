控制系统与频域分析基础

第一章：控制系统与频域分析基础

1.1 线性系统分析基础

传递函数
传递函数是在拉普拉斯变换域中，系统输出与输入之比。它是描述滤波器等线性时不变系统特性的核心数学模型。

零点与极点

  • 零点：使传递函数分子为零的复频率值。
  • 极点：使传递函数分母为零的复频率值。在伯德图的幅频特性曲线上，每个极点会使曲线的渐近线斜率下降 20 dB/dec（每十倍频）。

增益裕量 GM
在系统相位达到 -180° 时（即相位交越频率点），系统的增益距离 0 dB 还有多少余量。

  • 工程意义：评估稳定性的双保险。如果只有相位裕量合格而增益裕量不足，当系统的物理元件参数（如电容 ESR、电感感量）随温度和寿命发生漂移，或者输入电压/负载发生剧烈跳变时，增益曲线可能整体上移，导致极易越过 0 dB 线发生高频震荡。
  • 性能判据：
    • GM < 0 dB（通常在图表上表现为负值）：系统稳定。
    • 工程建议：通常要求增益裕量至少为 -10 dB 甚至更低（即绝对值大于 10 dB），以保证系统在全生命周期和全温区内的绝对稳定。

标准补偿器拓扑与传递函数

模拟补偿器基础与通用模型
模拟补偿器是模拟电源控制中的核心环节，其本质是控制理论在模拟电路中的硬件形态实现。从功能上讲，模拟补偿器利用运放、电阻和电容构建特定的零极点网络，以调整环路增益和相位，确保系统稳定。

在数字控制中，控制算法通过代码实现（如 PI、PID、2P2Z、3P3Z）；而在模拟控制中，这些算法则转化为物理电路结构，常见于 UC3843、LM5021 等控制芯片的外围补偿网络。

ZPK 模型在硬件工程中的物理意义

硬件电路（PCB）由电阻（R）和电容（C）构成，而 的物理单位即为时间（秒），这与时间常数 天然等同。因此，获取零极点时间常数 和 后，即可通过 平滑反推出具体的阻容值。
在如 800V / 30kW 级别的大功率电力电子系统控制环路中，零点、极点与增益分别承担着极其严苛的物理职责：

  • 零点（Z）——“预判机制”：对应微分动作，对误差的变化率极其敏感。物理表现为高通属性，频率越过零点后，增益曲线以 斜率上升，并提供最大 的相位提前。当输出电压出现跌落趋势时，零点能提前增加占空比进行补偿。在 Type III 补偿器中，外围 LC 滤波器会带来 相位延迟，必须靠配置两个零点强行拉回相位以保证裕度。
  • 极点（P）——“减震器”：对应积分或低通滤波动作，对高频突变反应迟钝。物理表现为低通属性，频率越过极点后，增益曲线以 斜率下降，并带来 的相位滞后。极点主要用于滤除开关频率带来的高频斩波噪声，并压制 ESR 零点，防止 PWM 比较器因高频干扰而发生抖动失稳。
  • 增益（K）——“油门”：对应比例动作。K 值本身不改变零极点位置或 Bode 图的曲线形状，而是将增益曲线整体上下平移。调整 K 值本质上是在调整系统的交叉频率（带宽）。K 值越大系统响应越快，但也更易受到高频干扰。

时间常数与起效频率的映射逻辑

这里存在一个经典的工程陷阱：时间常数 变大并不代表零极点的“属性变强”，而是代表其“起效频率变低”。
零极点的强度在 Bode 图上是固定的（永远是 和 ）。时间常数决定的是零极点的出场时机：

  • 极大时：计算出的频率 极低。这意味着该零极点在很低的频段即被激活并开始发挥作用。
  • 极小时：计算出的频率 极高。这意味着该零极点需等到极高频段才起作用，对中低频段的控制环路几乎无影响。
    综上，在重型补偿器（如 Type III）设计中，我们的终极目标是：精准地将零点配置在 LC 谐振频率附近以提前发力补偿相位，并将极点配置在 ESR 零点及开关频率附近以迅速压制高频噪声。

环路带宽设计与物理限制

带宽设计的核心考量
在模拟电源控制系统中，闭环带宽（ ）虽然由设计者通过外部补偿元件主动设定，但在实际工程中，其设计值不能无限增加。追求极高的响应速度必须受到系统物理极限的制约。带宽的设计主要受到功率级拓扑、采样定理以及控制器件自身性能的严格限制。

物理限制因素

  1. 开关频率限制
  • 物理机制：根据奈奎斯特采样定理及稳定性要求，环路带宽必须远低于系统的开关频率 。若带宽过高，系统会将开关纹波误判为反馈误差信号并进行剧烈调节，从而可能引发混沌、占空比抖动或次谐波振荡。
  • 工程设计边界：建议目标带宽 通常设计为开关频率的 至 。例如，若开关频率 ，则目标带宽最高应限制在 至 之间。
  1. 右半平面零点限制
  • 物理机制：对于 Boost、Buck-Boost 及其衍生拓扑（如反激 Flyback），在连续导通模式（CCM）下，其传递函数中存在右半平面零点（RHPZ）。与提供相位超前的左半平面零点不同，RHPZ 在提升增益的同时，会引入严重的相位滞后（最多可达 ），且这种滞后无法通过常规补偿器消除。
  • 工程设计边界：带宽设计必须远远避开 RHPZ 发挥作用的频率区域。工程经验要求，带宽 应设定为 RHPZ 频率 的 至 。例如，若计算得出 ，那么无论开关频率有多高，带宽设计的绝对上限也必须控制在 至 范围内，否则系统极易发生振荡。
  1. 误差放大器的自身性能限制
    控制芯片内部误差放大器（EA）的处理能力构成了器件级的性能极限，这可从大信号和小信号两个维度进行分析。
  • 增益带宽积（GBW）限制（小信号）：运放的 GBW 决定了其在高频下能提供的最大增益。在设计闭环带宽 时，该频率点上补偿器所需的高频增益，绝不能超过误差放大器自身的开环增益曲线。在常规电源设计中，除非涉及兆赫兹级的高频应用，否则 GBW 通常不会成为主要瓶颈。
  • 压摆率（SR）限制（大信号）：压摆率是指运放输出电压在单位时间内能够变化的最大速率，单位通常为 。在 30kW 级别的大功率系统中，这一点至关重要。当系统发生大范围负载跳变（如从满载瞬间切换至空载）时，运放输出端（COMP 引脚）的电压需要瞬间大幅拉低以减小占空比。如果运放的 SR 不足，即使小信号带宽（GBW）设计得再高，COMP 电压也无法快速下降，这将导致输出电压出现极其严重的过冲，甚至可能损坏后级负载。因此，大信号瞬态响应的瓶颈往往受限于运放的 SR。

环路补偿标准化设计流程
基于上述限制条件，模拟电源环路补偿的标准化工程设计流程如下：

Step 1：确定目标带宽上限
首先计算并比较各项限制因素，取其中的最小值作为最终的设计目标。

  • 依据开关频率计算限制值：$\text{Limit}1 = \frac{f{sw}}{10} \sim \frac{f_{sw}}{5}$
  • 依据右半平面零点计算限制值（仅适用于 Boost/Buck-Boost 族系拓扑）：$\text{Limit}2 = \frac{f{RHPZ}}{4} \sim \frac{f_{RHPZ}}{3}$
  • 最终目标带宽 ：取 与 中的较小值。

Step 2：计算补偿参数
根据确定的目标带宽 和期望的相位裕度（PM，通常设定为 ），结合功率级的传递函数，使用 Mathcad、MATLAB 或其他环路计算工具求解补偿网络的元件参数。

  • 输入：功率级参数（L、C、ESR 等）、目标 、目标 PM。
  • 输出：计算出所需的零点时间常数 和极点时间常数 ，进而反推出补偿网络中电阻 和电容 的具体数值。

Step 3：硬件物理实现与验证
将计算得到的电阻和电容元件焊接至控制芯片的 COMP 引脚及相关反馈回路中。此时，电源系统的物理带宽和动态响应特性即被硬件参数固定在设计目标值上。最后，通过网络分析仪（如 Bode 100）进行扫频测试，验证实际的增益裕度（GM）和相位裕度（PM）是否符合预期。

控制策略匹配与工程变体电路

控制模式与补偿器的匹配逻辑
选择 Type II 还是 Type III 补偿器，并非盲目套用，而是严格取决于被控对象，即功率级的频率响应特性。补偿的最终目标，是使功率级与补偿器构成的总开环增益，在穿越 0 dB 频率点时，呈现理想的单极点滚降特性，即 -20 dB/dec 的斜率，以保证充足的相位裕度。

  • 电压模式控制 (VMC) 与 Type III 补偿

    • 被控对象特性：输出端由电感 L 和电容 C 构成一个二阶系统。
    • 频率响应：在 LC 谐振频率处，增益会以 -40 dB/dec 的斜率急速下降，同时相位会发生急剧的 -180° 滞后。
    • 匹配策略：必须使用 Type III 补偿器。其提供的两个零点能够产生最大 +180° 的相位提升，用以抵消 LC 双极点带来的相位骤降，从而将穿越频率处的斜率修正为 -20 dB/dec，确保系统稳定。

  • 电流模式控制 (CMC) 与 Type II 补偿

    • 被控对象特性：由于电感电流被内部高速电流环（内环）精确控制，在小信号模型中，电感表现为一个受控电流源。系统因此降阶为由输出电容 C 和负载 R 构成的一阶系统。
    • 频率响应：增益以 -20 dB/dec 的斜率平缓下降，相位主要滞后 -90°。
    • 匹配策略：使用 Type II 补偿器即可。它提供的一个低频零点所带来的 +90° 相位提升，足以将系统的相位裕度调整至安全范围。

特例说明

  • DCM 模式降阶：在断续导通模式 (DCM) 下，即使是电压模式控制的功率级，也会因电感电流在每个周期归零而表现出一阶系统特性。此时，系统退化，可以使用 Type II 进行补偿。
  • 强抗噪需求：在某些高噪声的 CMC 应用中，可能会在 Type II 基础上增加元件，构成类似 Type III 的拓扑。其主要目的并非获取额外的相位提升，而是利用新增的高频极点来更强力地抑制开关噪声。

工程应用中的变体补偿电路
在实际工程中，除了标准的电压型运算放大器，还有两种非常主流的物理实现形式。它们在电路接法上与标准运放差异显著，但在数学本质上完全等效于 Type II 或 Type III 补偿器。

  1. 跨导放大器 (OTA) 补偿
    常见于集成电源管理芯片，如 LM2596 及许多现代 PWM 控制器。
  • 物理特征：使用跨导放大器 ，其输出为电流而非电压。因此，补偿网络连接在 COMP 引脚与地 (GND) 之间，而非跨接在运放的输入与输出之间。
  • 等效类型：通常配置为等效 Type II。
  • 阻抗网络：由电阻 与电容 串联，再与高频滤波电容 并联构成。
  • 传递函数：
  • 工程特点：其数学形式与电压型 Type II 补偿器高度一致，实现了完全相同的零极点配置，但外围布线更为简洁。
  1. TL431 与光耦隔离补偿
    在反激 (Flyback) 和各类隔离型辅助电源设计中占据主导地位。
  • 物理特征：利用 TL431 内部的基准和高增益放大器作为误差放大器，通过光耦实现原副边的信号隔离传输。电路中没有常规意义上从运放反相端到输出端的 RC 反馈回路。
  • 等效类型：绝大多数情况下被配置为 Type II 补偿。
  • 关键参数：
    • ：输出电压的上拉分压电阻，决定了补偿器的输入阻抗。
    • ：连接在 TL431 阴极 (Cathode) 与参考极 (Ref) 之间的补偿阻容网络。
    • ：光耦支路的综合增益因子，由光耦的电流传输比 (CTR)、原边限流电阻 和副边上拉电阻 共同决定。
  • 传递函数：
  • 工程痛点：除了 RC 参数的整定，光耦的 CTR 会因温度剧烈变化和元件老化而产生大幅漂移。这会直接导致环路整体增益发生改变（表现为 Bode 图的整体上下平移），是隔离电源环路设计中最难以控制的因素之一。因此，设计时必须留出充足的增益裕度。

其他控制策略概览
为了构建完整的控制理论认知体系，除上述模拟线性控制外，电力电子领域还存在以下控制策略：

  • 数字线性控制：通过 DSP 或 MCU 在 z 域利用差分方程实现，如 2P2Z (等效于模拟 Type II)、3P3Z (等效于模拟 Type III)。广泛用于服务器电源、车载充电器 (OBC) 和光伏逆变器。
  • 非线性控制：如滞环控制 (Hysteretic)、恒定导通时间控制 (COT)。这类控制没有固定的开关频率，无需复杂的传统 RC 补偿网络，具有极快的瞬态响应速度，常用于对动态电流要求苛刻的 CPU/GPU 核心供电 (VRM)。
  • 基础理论模块：PID 控制、超前-滞后补偿 (Lead-Lag)。这些是经典控制理论的基础。电力电子工程师常说的 Type II/III 补偿器，正是这些基础理论在硬件电路上的具体实现。
