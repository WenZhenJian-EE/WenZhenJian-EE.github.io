从零极点到实战：Type II/III 模拟补偿器设计与调试

做开关电源或者任何闭环控制系统，环路调试都是决定动态响应和稳定性的关键。如果电网或负载突变时补偿网络搭得不好，系统就会产生漫长的振荡，甚至直接自激，听到电感发出啸叫声，接着就是磁芯饱和、管子“啪”的一声炸掉。

前阵子调一款 500W 的车载 DCDC，就在环路自激上吃了不少苦头。今天把 Type I、II、III 模拟补偿器的物理直觉、阻容参数计算步骤以及实验室 Bode 扫频调试的心得整理分享出来。

1. 环路补偿的本质：带宽与相位的拉锯战

我们在实验室调环路，最理想的状况是：负载从 10% 瞬间跳变到 90% 时，输出电压能以最快的速度恢复，而且波形不能有大起大落的超调。

  • 带宽（穿越频率 ）：就是开环增益到 0dB 的频率。带宽越宽，响应负载阶跃的速度就越快。
  • 相位裕量（PM）：在穿越频率 处，相位距离 （失控自激的死线）的安全余量。裕量越大，系统越稳定，超调越小。一般工程上合格线是 。
  • 增益裕量（GM）：在相位跌到 时，增益必须足够小（建议低于 ），防止元件温漂和老化导致自激。

1.1 实验室里的自激现场

当时那台 OBC 样机，满载切轻载时，输出电压产生了长达 5 个周期的衰减振荡，超调量超过 8%。用 Bode 100 扫频一测，发现虽然 调到了 ，但相位裕量只有可怜的 25°。这说明系统相位储备严重不够。必须引入额外的零点，在 附近把相位强行“托起来”。

2. 三种模拟补偿器（Type I/II/III）的拓扑与直觉

模拟补偿器就是利用运放的虚短虚断，通过输入和反馈端的阻容组合，搭出我们想要的传递函数：

2.1 Type I 补偿器：纯积分器（I 控制）

  • 电路：输入是单颗电阻 ，反馈是单颗电容 。
  • 零极点：只在原点 处有一个积分极点。
  • 直觉：它提供永久的 相位滞后，没有任何相位提升能力。直流增益极高（用来消除稳态静差），但增益以 一路滚降。
  • 应用：只适用于低频控制、本身相位裕量极其充沛但响应极慢的系统。比如 PFC 的电压外环，带宽一般就 。

2.2 Type II 补偿器：PI 控制 + 高频极点

  • 电路：反馈端是用电阻 串联电容 ，再并联一个高频滤波小电容 。
  • 零极点：1 个原点极点（），1 个低频零点（），1 个高频极点（）。
  • 直觉：
    • 低频零点 提供最高 的相位超前，负责把相位拉起来，对冲功率级电容或电感的滞后。
    • 高频极点 负责压低高频开关噪声，防止运放饱和。
    • 它在中间频段能托起一部分相位，但最大提升相位小于 。
  • 应用：适用于电流模式控制（一阶系统，如 CMC 的 Buck）或者输出电容 ESR 很大的系统。

2.3 Type III 补偿器：PID + 双高频极点

  • 电路：在 Type II 的基础上，在输入电阻上并联一个前馈阻容支路。
  • 零极点：1 个原点极点，2 个零点，2 个高频极点。
  • 直觉：
    • 双零点合力能提供最多 的超前相位！可以精准对冲二阶系统（电压模式控制下 LC 共轭谐振）产生的断崖式 相位突降。
    • 双高频极点用来死死卡住高频开关纹波。
  • 应用：电压模式控制（VMC）的二阶 LC 系统，或者严重缺相位的谐振拓扑（如 LLC）。

3.1 零极点放置黄金法则

  1. 打平谐振峰：把双零点 放置在 LC 谐振频率 附近，抵消共轭极点带来的增益峰和相位突变。
  2. 抵消 ESR 零点：把第一个高频极点 放在电容的 附近，防止高频增益昂扬。
  3. 压住开关噪声：把第二个高频极点 放在开关频率 的一半（奈奎斯特频率）附近，加速高频滚降。
