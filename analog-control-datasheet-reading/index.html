为什么UC384x老树发新芽：BiCMOS工艺的UCC28C4x有什么料？

我们在做反激或者正激辅助电源时，经典的UC384x系列基本上是绕不开的常客。而UCC28C4x则是TI针对它推出的BiCMOS工艺升级版。

为什么我们要舍得花高价升级到BiCMOS工艺？因为这种工艺把双极型晶体管（Bipolar）的强驱动能力、高响应速度，和CMOS工艺的低功耗集成度完美结合了。最直接的表现是，它的启动电流降到了典型值50μA，这在超宽压输入启动设计或者降低待机损耗时简直是救命稻草。同时，它的输出驱动级依旧保留了双极型的强悍输出，提供±1A的驱动能力，哪怕开关频率推高，也能轻松拿捏。

在实际选型中，这几个子型号对应的UVLO（欠压锁定）和占空比限制，是我们画板子和搭系统时必须死死咬住的避坑点：

  • UCC28C42 与 UCC28C44：开启电压14.5V，关闭电压9V。这5.5V的宽滞回窗口非常好用，能让我们在AC-DC离线式或高压直流输入的辅助电源设计中，选用更小的启动电容，同时完美避开上电时因为VDD电压跌落引起的启动振荡。
  • UCC28C43 与 UCC28C45：开启电压8.4V，关闭电压7.6V。这组低开启电压版本非常适合用在低压DC-DC变换器（如常见的12V转5V辅助源）。
  • UCC28C40 与 UCC28C41：开启电压7.0V，关闭电压6.6V。这个版本更低，主要用于电池供电的便携式设备中。
  • 占空比硬限制：UCC28C40/42/43 的最大占空比能飙到100%，反激拓扑一定要选它们；而 UCC28C41/44/45 内部被硬件限制了最大占空比在50%以内，这是为了给正激拓扑提供硬性的磁芯复位时间，防止高占空比下直接变压器偏磁炸管。

UCC28C4x引脚功能与我们调试时的重点

芯片通常采用经典的SOIC-8或VSSOP-8封装，引脚定义如下：

引脚 名称 功能描述 关键注意事项
1 COMP 误差放大器输出 环路补偿的命门。隔离设计里经常接光耦拉低，直接强拉到低电平（如低于0.1V）就能让占空比瞬间归零，实现快速软保护。
2 FB 电压反馈输入 运放反相输入端，内部自带2.5V参考。我们做次级反馈的隔离电源时，通常直接把FB脚接地。
3 CS 电流检测输入 接到开关管源极的采样电阻上。注意内部过流限流阈值死死卡在1V，这脚一定要加RC滤波，不然开关尖峰会提前误触发过流。
4 RT/CT 振荡器定时 接定时电阻电容到VREF和GND，用于调节工作频率和最大占空比。
5 GND 地 信号地和功率地的合路端。画板时（PCB Layout）千万注意大电流功率走线不要经过控制地，单点接地是死命令。
6 OUT 驱动输出 经典的图腾柱结构，提供±1A的峰值电流，可直接驱动常规的开关管MOSFET。
7 VDD 电源输入 绝对最大耐压是20V。我们在调试时，一定要在VDD对地并一颗18V左右的稳压管，不然辅助绕组的漏感尖峰很容易击穿芯片。
8 VREF 基准电压输出 输出精密的5V基准，必须加旁路电容滤波，也能给外围微功耗电路充当参考源。

我们在研发中的外围设计与参数计算心得

以我们在研发中做过的一款经典的12V/48W离线式反激辅助电源为例，分享几个外围电路的调试和避坑经验：

  • 启动电路的设计：启动阶段由于主变压器还没工作，VDD电解电容的电全靠启动电阻灌进来。因为BiCMOS工艺的启动电流只有50μA，我们可以把启动电阻做大以降低待机功耗。推荐选用420kΩ，但千万不要只用一颗贴片，建议用两颗210kΩ串联，既能分担耐压也能防止单个电阻老化击穿开路。
  • VDD供电回路：推荐使用47μF到120μF的电解电容并联一颗0.1μF的陶瓷电容。电解电容容值如果小了，主绕组供电还没完全建立起来，VDD电压就已经跌过UVLO关断阈值了，系统就会出现打嗝重启的毛病。还有，绝对别忘了并联一颗18V稳压管，这芯片VDD极限耐压才20V，漏感尖峰大一点就炸管。
  • 振荡器定时元件：频率设定电阻接在VREF和RT/CT之间，电容接在RT/CT与地之间。如果设定频率为110kHz，推荐定时电阻选用15.4kΩ，电容选用1nF。根据经验，工作频率估算公式为：。实际调试时，频率可能会有小幅漂移，需要根据实际寄生参数微调电阻。
  • 电流采样电路（CS）：采样电阻阻值通过公式计算：。如果原边设计峰值电流是2A，电阻选0.5Ω。这里有一个关键的避坑点：一定要在采样电阻和CS引脚之间加一个RC低通滤波器（通常是1kΩ3.8kΩ电阻配合100pF330pF电容），滤除MOS管开通瞬间由于结电容放电和二极管反向恢复引起的前沿尖峰，否则系统一开机就会因为尖峰误触发限流而带不起载。
  • 隔离反馈（次级控制）：在隔离反激拓扑中，我们一般在次级放TL431和光耦（PC817等）来做环路控制。初级侧直接把FB脚接地，让芯片内部误差放大器当电流环跟随器。光耦集电极拉到COMP上，上拉电阻推荐10kΩ到20kΩ接VREF，发射极直接接地。
  • 栅极驱动阻尼：OUT引脚推MOS管时，千万别直连。串联一颗10Ω~22Ω的栅极电阻，不仅能抑制高频寄生振荡，还能调节MOS管的开关速度，用来在EMI指标和开关损耗之间做平衡。
  • 基准旁路滤波：VREF脚的0.1µF~1µF陶瓷电容一定要紧贴芯片引脚接地，保证基准源的绝对洁净。

内部数模混合架构与工作机制剖析

我们在调试模拟电源时，如果搞不清楚芯片内部的工作逻辑，很容易在遇到异常波形时抓瞎。这芯片内部是典型的数模混合结构：

  • 误差放大器：一个标准的双极型差分输入运放。将FB电压与内部2.5V参考进行对比，在COMP引脚输出代表当前系统功率需求的连续变化模拟电平。
  • PWM比较器：同相端接CS的电流斜坡，反相端接COMP的调理信号。当电流斜坡涨到比COMP设定的阈值高时，比较器翻转，输出高电平准备关断开关管。
  • 信号调理与1V硬钳位：注意，COMP电压并不是直接送入PWM比较器反相端的。为了扩宽控制范围和抗干扰，它会先减去约1.4V（两颗二极管压降）再经过三分之一分压，公式为：。同时内部还有一个1V的硬件钳位。这意味着，哪怕COMP电压被拉得再高，送到比较器的反向端最高也只有1V。这也就是为什么我们的限流阈值是1V的物理根源。
  • SR锁存器：这是防止开关噪声引起误触发的护身符。它实行‘定时置位，撞线复位’的逻辑。时钟脉冲周期性把S端置位，OUT输出高电平开启MOS管；一旦CS锁死，比较器输出翻转拉高R端将锁存器复位，OUT立刻拉低。在一个周期内，锁存器锁死只允许触发一次，这彻底避开了在门槛附近因为噪声引起的多次开关（双脉冲/多脉冲）隐患。
  • 欠压锁定（UVLO）：自带滞回的比较器。以UCC28C42为例，开启14.5V，关闭9V。这5.5V的宽度，能让电源在刚启动、辅助绕组尚未接管的过渡期，靠电解电容的残存电量多撑一会儿，保证平稳建压。
  • 或门保护逻辑：只要出现UVLO欠压、基准电源未就绪或SR锁存器关断信号的任一情况，或门都会瞬间输出高电平，通过下拉三极管把OUT引脚强行拉到地，保护后级功率管。

峰值电流模式的双环控制哲学

UCC28C4x采用的是经典的峰值电流模式控制。它的内部由两个环路嵌套而成：

  • 电压外环（负责下达指令）：通过次级采样并经过光耦，在COMP引脚输出当前系统所需的电流阈值。它不直接控制占空比，而是动态调整内环的限流门槛。
  • 电流内环（负责动作执行）：每个开关周期开始时强制MOS管导通，电感电流呈斜坡爬升，CS检测该斜坡。一旦CS电压撞到了电压外环设定的COMP门槛，电流内环立即动作，掐断占空比。
  • 逐周期限流保护：在过载或输出短路时，电压外环会拼命把COMP拉高，但由于内部1V硬钳位的限制，电流内环的上限被死死锁死在1V，从而强行关断MOS管，保证每个周期电流都不会超过设计最大值，避免磁饱和炸机。

这种“定时开通，撞线关断”的电流内环带来了天然的物理优势：电压前馈响应极其迅速。当输入母线电压突然波动时，电感电流上升斜率变陡，CS引脚能以更快速度撞线关断，在电压外环还没来得及反应之前，在同一个开关周期内就缩短了占空比，极大减轻了系统动态调整的压力。

模拟控制与DSP数字控制的概念映射

为了让我们做数字控制研发时能互通，这里把UCC28C4x的模拟控制逻辑与C2000 DSP的数字控制概念做了一个映射对照表：

C2000数字控制概念 UCC28C4x模拟控制映射 本质功能
Reference (参考变量) 内部2.5V VREF 目标基准电压
ADC采样反馈结果 FB引脚电压 实际状态反馈值
PI控制算法代码 运放及外部阻容补偿网络 误差放大与动态调节
PI计算结果 (如CMPA值) COMP引脚电压 当前周期的目标控制量（阈值）
PWM定时器计数值 (TBCTR) CS引脚电压斜坡 随时间累积的实时运行量
TBCTR匹配CMPA触发动作 CS电压超越COMP电压触发翻转 决定导通时间（占空比）的终点

这两者在生成占空比上有个本质区别。数字控制是“预判型”，控制器根据上一周期的计算结果直接写入寄存器，占空比在周期开始前就是确定的；而模拟峰值电流控制是“实时反应型”，导通是时钟触发的，但关断完全由电感电流斜坡和电压反馈电平在比较器上现场“撞线”决定。

SN6505x-Q1推挽变压器驱动器核心参数扒皮

我们在设计数字隔离接口电源（比如给CAN/RS485隔离收发器供电的5V-5V小电源）时，SN6505x-Q1是非常好用的片子。但是在选型和设计中，这几个参数决定了硬件的鲁棒性：

开关频率（Fsw）

  • 定义与名词：手册里的 指芯片内部MOSFET的交替开关频率。有A版本（160kHz）和B/D版本（420kHz）。要注意的是，由于它是互补推挽工作，每个MOS管的实际开关频率是 。
  • 我们在设计时的权衡：
    • 高频 (420kHz)：允许使用更小尺寸的微型变压器和输出电容，省板子空间和BOM成本。但代价是开关损耗大，芯片本身发热明显。
    • 低频 (160kHz)：开关损耗小，效率高。但是需要磁感值更大的变压器，体积会变大。
    • EMI性能：A版本（160kHz）虽然频率低，但它自带扩频时钟（SSC）功能，能把能量从单点打散到频带里，在过EMI测试时很容易扫过去；而420kHz版本高次谐波多，需要我们在PCB布局和滤波设计上多花心思。

开关管导通电阻（）

  • 定义：芯片内部两个MOSFET完全导通时的源漏电阻。在 下，典型值0.16Ω，最大值0.25Ω。
  • 物理影响：它决定了传导损耗 。同时，它会在初级侧产生一个电压降 。这在高阻抗或者低输入电压（比如3.3V转5V）重载应用中尤为明显。如果 过大，实际加在变压器上的电压被扣除太多，次级输出电压会严重下跌。
  • 设计建议：设计时一定要按全温范围下的最大 算热损耗。另外，芯片的D1、D2和GND引脚是发热大户，PCB走线必须短粗，多打孔接到内层大面积地平面上散热。

电流极限值（ILIM）

  • 定义：芯片能够驱动的物理最大电流上限。达到这个值，芯片会切断驱动以保护内部功率管和外部变压器。 时典型值是1.75A。
  • 设计防线：在做隔离5V/1A大功率输出时，初级侧的平均电流已经到了1.25A左右（按80%效率算），推挽的瞬间峰值电流极易踩到 ILIM。所以设计时，正常工作的最大峰值电流至少要比 ILIM 的最小值（1.42A）留出20%~30%的裕量。
  • 软启动时间（tss）：上电时给次级大电容充电会产生很大的浪涌电流。A/B版本自带软启动，能平滑建压；而D版本为了追求极速上电，去掉了软启动，如果你的次级电容很大（如 >100µF），上电瞬间就会被 ILIM 掐断导致启动失败。

当我们面对一个新的变压器驱动芯片时的选型指南

以后我们在做类似推挽、半桥隔离驱动源时，直接用这套清单去扒数据手册：

  1. 拓扑与集成度评估：是推挽还是桥式？内部集成了MOSFET还是纯控制器？（如SN6505就集成MOSFET的推挽，外围极简）。
  2. 核心电气参数：工作电压范围 能不能覆盖我的电源？开关频率 是多少，带不带扩频，能不能外部时钟同步？导通电阻 是多少，大电流下压降厉不厉害？
  3. 保护与可靠性机制：过流保护（OCP）是打嗝还是逐周期？UVLO阈值带不带足够的迟滞？有没有热关断（TSD）？带不带软启动？
  4. 封装与热阻：热阻 是多少，大电流下结温会不会超标？底部带不带散热裸垫？

古董芯片TL494的独特反馈设计

虽然TL494非常老，但在许多低成本推挽或半桥电源中，它依然非常受欢迎。它的控制逻辑非常有个性，和我们常见的UC3842等芯片正逻辑（误差放大器输出越高，占空比越大）相反：TL494的内部硬件逻辑是，误差放大器输出电压越高，输出的PWM脉宽越窄，即占空比越小。

这在设计负反馈电路时就带来了一个很有意思的接法：

  • 正压反馈接法：为了适配这个反向逻辑，输出电压反馈信号必须接在放大器的同相输入端（+），而参考基准接在反相端（-）。当输出电压偏高时，反馈信号增大，同相端电位上升，放大器输出升高，占空比阻断拉小，顺利拉回输出电压，形成正确的负反馈。
  • 负压采样妙用：如果我们做某些为了省成本的设计，利用地回路线圈上的电阻做负电压采样。同相端（+）直接接地，反相端（-）接一个分压求和网络。随着电流增大，采样点产生更深的负电位，把反相端往地电平以下拽。当反相端电位低于同相端（GND）时，放大器输出上升，从而拉小占空比。这种“负负得正”的负反馈接法非常巧妙，是模拟控制逻辑的一种经典应用。
