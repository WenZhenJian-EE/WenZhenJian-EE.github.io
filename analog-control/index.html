模拟闭环稳定性：次谐波震荡数学机理、NPN斜坡补偿电路、全桥驱动方案、TL494与SG3525选型对比及数模混合峰值电流实现

在电力电子的模拟控制领域，峰值电流控制（Peak Current Mode Control, PCMC）凭借其极佳的动态响应和天然的逐周期过流保护，成为了开关电源（如 Buck、反激、正激）的统治级控制算法。

然而，当占空比超过 50% 时，PCMC 会触发著名的“次谐波震荡（Subharmonic Oscillation）”。

本篇手记深度剖析次谐波震荡的数学推导、如何在老一代芯片（如 UC3842）上通过分立三极管搭建斜坡补偿、经典芯片（TL494 与 SG3525）的选型硬伤，并揭秘现代 MCU 如何通过数模混合外设（DAC + 比较器 + 硬件封锁）实现真正的峰值电流控制。

一、 次谐波震荡的数学推导与物理本质

次谐波震荡表现为相邻两个开关周期的占空比一大一小，电感电流波形呈现高低交替的大小波，伴随变压器啸叫，且极易在长周期突发大电流导致功率管过温炸毁。

1. 经典 UC3842 分立三极管斜坡补偿硬件电路

对于没有内置补偿的经典模拟芯片，我们需要利用芯片原有的定时电容 RT/CT 上的锯齿波来调制电流采样引脚 ISENSE。

  • 工作机制：
    1. NPN 三极管 Q1 组成射极跟随器（Buffer）：基极引脚直接阻抗极高地接入 RT/CT 引脚。利用三极管的跟随特性，将定时电容上的 0V~2.7V 锯齿波电压镜像到发射极上，同时实现单向隔离，防止从敏感的振荡引脚抽取电流而破坏工作频率。
    2. 电阻加法网络：三极管发射极的锯齿波通过斜坡电阻 送往 ISENSE 引脚，与电感电流在采样电阻 上的压降，通过 ISENSE 滤波电阻 构成电阻式电压加法器。

三、 经典模拟控制器选型决断：TL494 与 SG3525 的硬伤对比

在非 DSP 控制的小功率逆变器或 DCDC 中，TL494 和 SG3525 是最长盛不衰的两款电压型 PWM 控制芯片。

  • 工程选型法则：
    • 反激拓扑：严禁使用 TL494/SG3525。反激在连续模式存在右半平面零点，电压单环动态极慢且易振荡，必须使用 PCMC（如 UC3842）以防止磁饱和。
    • 车载逆变升压推挽段：选用 SG3525。利用其自带的推挽驱动能力直接驱动门极变压器，简化外围电路。
    • 大功率电池恒流恒压充电器：选用 TL494。利用其双误差放大器的并联竞争机制实现高精度电压电流切换。

四、 现代 MCU 数模混合峰值电流控制的闭环机制

在单片机（如 C2000 DSP、STM32G4）数字电源中，如果纯靠软件读取 ADC 再用 PI 计算出占空比，这属于平均电流模式控制（ACMC）。由于软件计算、ADC 转换存在数微秒的滞后，数字控制无法直接在纳秒级实现峰值截断。
为了实现真正的峰值电流限制，现代 MCU 采用数模混合外设架构（如 C2000 的 CMPSS 模块）：

  1. 参考下发：软件电压外环运行 PID 算法，将计算出的峰值电流限制指令写入内部高精度 DAC 寄存器，由 DAC 输出动态的模拟基准电压给内部高速比较器的负端。
  2. 高频比较：将实测电感电流信号引入比较器的正端。CMPSS 模块内部集成了数控斜坡发生器，能在片内直接完成模拟斜坡补偿叠加。
  3. 硬件封锁：当电流波形触碰 DAC 设定的天花板时，内部比较器输出跳转，直接触发 PWM 模块的 Trip Zone（错误控制区）。硬件逻辑在几纳秒内直接强行拉低 PWM 驱动电平。
  4. 整个截断过程完全不需要 CPU 软件介入，响应速度与模拟芯片持平，同时具备数字外设在线调整阈值的极大灵活性，代表了现代高频电力电子控制器的最高技术水准。
