峰值电流模式控制与次谐波震荡原理

在开关电源的峰值电流模式控制中，当占空比大于 50% 时，系统会发生次谐波震荡。其宏观表现为相邻两个开关周期的占空比一大一小，电感电流波形出现明显的大小波，常伴随啸叫声并容易导致开关管损坏。

该现象源于峰值电流控制的底层逻辑。与普通电压控制模式相比，其差异在于：

  • 普通电压控制模式：系统遵循固定的开通与关断时间，如同按秒表执行任务。即使受到干扰导致电流起点偏高，系统仍会维持既定的时间周期，波形仅整体平移，不会引发震荡。
  • 峰值电流控制模式：系统通过监测电感电流，一旦其触及给定的参考阈值 就立刻关断开关管。在此逻辑下，占空比不再是固定值，而是电流波形与阈值动态作用的结果。

斜坡补偿理论与硬件级实现

单相全桥逆变器

做单相全桥逆变器不一定需要单片机，根据输出波形需求，有多种专用芯片和模拟芯片方案可供选择。

纯正弦波逆变器方案：推荐使用专用 SPWM 发生芯片，如 EG8010、TDC0820、HT2201。这类芯片内部固化了逻辑，通过调整外部引脚电平即可设置 50Hz 或 60Hz 频率、死区时间和软启动。若纯粹为了验证原理，也可使用分立模拟元件（如 LM324 产生 50Hz 正弦参考波，结合高频三角载波输入到 LM393 模拟比较器）搭建 SPWM，但因温漂和老化问题目前仅限教学实验。

方波或修正正弦波方案：适用于对波形要求不高的负载，经典方案为 SG3525、KA3525 或 TL494 等双路输出 PWM 芯片。

无论采用单片机还是模拟控制芯片，弱电控制信号都无法直接驱动全桥上桥臂源极浮动的功率管，必须在控制芯片与全桥功率管之间加入半桥驱动芯片（如 IR2110、IR2113 或国产替代 EG2113），利用其自带的自举电路放大控制信号并实现浮地高压驱动。

在方波或修正正弦波全桥逆变器中，对角线的开关管需要同时导通。因此，使用 TL494 或 SG3525 这类具有两路互补 PWM 输出的模拟芯片即可驱动全桥拓扑。实现浮地驱动的常见工业做法有两种：

  • 使用半桥驱动芯片：例如两片 IR2110，将控制芯片的两路 PWM 信号交叉连接到左桥和右桥的高侧与低侧输入端，在设定好死区时间确保全关断的前提下实现对角导通，从而避免全桥上下管直通损毁。
  • 使用门极驱动变压器：通过物理变压器磁耦合实现高侧管驱动，该方案抗干扰能力极强且无需自举电容。SG3525 因自带推挽输出级，常被直接用于驱动此类小型门极变压器。

相比模拟芯片将对角管信号绑定的方案，现代 DSP 或高阶单片机控制全桥时通常采用四路完全独立的 PWM，其原因在于高级数字控制系统有更严苛的调制与保护需求：

  • 移相全桥等软开关拓扑中对角管并非同时导通，必须精确控制左右桥臂相位差以实现零电压开通。
  • 单片机内部死区发生器可针对不同桥臂的非理想特性独立、精准地插入纳秒级死区时间。
  • 单极性调制需要工频桥臂与高频桥臂在截然不同的开关频率下独立运行。
  • 独立的故障保护机制可在检测到过流时迅速仅封锁特定桥臂，使另一桥臂进入续流状态以保护功率器件并快速恢复。

单极性调制在负载两端实际表现为软件构建的三电平输出。传统的双极性调制仅在 和 之间剧烈切换，而单极性调制通过让同侧上下桥臂交替导通，在周期中插入了一个负载短路续流的零电平阶段。这种调制方式使开关管动作次数减半，同时将电压跳变幅度降至一半，使得输出电流纹波极大减小，有利于缩小滤波电感体积与磁芯损耗。此外，若采用带有中性点的真正硬件三电平（如 NPC 拓扑）配合单极性调制，可以在负载端输出 、 、、 、 共五个电平状态，波形更加逼近理想正弦波。

TL494 与 SG3525 同为经典的双路电压型 PWM 控制芯片，但其内部架构差异决定了它们各自不同的最佳适用场景。

典型适用场景 半桥电源、双向降压升压拓扑、多路输出控制或充电器等需要复杂反馈逻辑的场合。 推挽电源、全桥拓扑的前级驱动、车载逆变器升压级等需要直接驱动开关管的场合。

在降压拓扑中，TL494 表现优异。通过将其输出控制端接地，可将双路输出并联配置为最高占空比达 96% 的单端输出模式，外接半桥驱动芯片或自举电路驱动 MOSFET，配合电阻分压反馈网络即可实现高精度的恒压输出。

在反激拓扑中，工业界极少采用纯电压控制的 TL494。反激变压器在连续导通模式下系统传递函数存在右半平面零点，表现为增加占空比时输出电压会发生瞬态跌落。电压控制模式对此动态响应极慢，补偿网络难以调节且极易引发自激振荡。此外，由于缺乏逐周期电流限制，变压器容易发生磁饱和导致开关管炸毁。因此，反激电源的绝对主流方案是采用峰值电流控制的 UC3842 或 UC3843 系列芯片。

使用纯模拟芯片制作并网逆变器不仅极其困难且存在极高风险，该方案已被现代工业淘汰。离网逆变器本质上是建立稳定电压的电压源，而并网逆变器则是跟随电网电压波形注入能量的电流源。纯模拟电路并网面临以下三大难以跨越的技术屏障：

  • 锁相环难度极高：必须利用模拟电路搭建精密的锁相环以实时追踪市电过零点和相位，一旦存在微小相位差，极小的电网内阻便会引发致命的数百安培倒灌环流损毁硬件。
  • 电流闭环控制复杂：必须采集高频电感电流，并使用模拟高频乘法器和复杂运放网络去跟随市电的正弦波形，此类电路受温漂和噪声干扰极为严重。
  • 难以实现防孤岛保护：纯模拟电路难以实现主动频率偏移法等高可靠性的孤岛检测算法，电网断电时无法在毫秒级时间内迅速切断输出，危及电网抢修人员安全。

基于以上原因，现代并网逆变器完全依赖 DSP 或高阶单片机的强大数字运算能力来实现精准的软件并网控制。

即使是正激等理论上不存在右半平面零点问题的变压器隔离拓扑，在实际工程中依然广泛采用峰值电流控制，其核心目的在于防止变压器偏磁饱和。正激变压器在开关闭合时单向励磁，断开时依靠复位绕组或双管结构进行磁复位。若开关管开通延迟不一致或占空比发生微小抖动，导致正向励磁的伏秒积略大于复位的伏秒积，直流剩磁便会在磁芯中逐周期累加。峰值电流控制通过逐周期监测初级电流，在电流异常陡增并触碰阈值的瞬间立刻关断开关管，强行终止当前周期的单向励磁，从而彻底解决了偏磁饱和问题。此外，对于所有含变压器的拓扑，峰值电流控制还具备极速过流保护能力和输入电压前馈特性，当输入电网电压发生剧烈波动时系统能自动提前响应，维持输出电压高度稳定。

电压控制模式与峰值电流控制模式在底层环路运行逻辑上有着本质区别。电压控制仅包含一个电压单环，芯片将输出电压的误差信号直接与内部固定生成的人造锯齿波进行比较以切断 PWM，系统对电感的实际电流状态一无所知。而峰值电流控制在硬件层面上实现了电压外环与电流内环的串联双环结构。其电压外环计算出的误差信号不再直接参与切波，而是转化为电流内环的指令参考值；电流内环直接利用采集到的真实向上攀升的电感电流波形与该指令参考值进行比较，当真实电流触碰设定的电压天花板时瞬间关断 PWM。

在单片机数字控制系统中，纯粹依靠软件代码编写的串联双闭环控制通常属于平均电流型控制，而非峰值电流控制。为避开开关管高频切换时产生的振铃噪声，数字电源的 ADC 采样触发点通常设定在 PWM 波形的绝对中心点，在连续导通模式下，此处的采样数值恰好等于电感电流的平均值。内环算法利用该平均电流进行 PI 计算得出占空比数值，再交给定时器硬件去产生下一周期的 PWM。纯软件控制的执行过程包含了采样、运算及写入寄存器的固有延迟，因此无法在纳秒级别完成峰值截断。

现代高阶单片机实现真正的峰值电流控制依赖于其内部的数字与模拟混合外设架构。其运作机制为：软件层面的电压外环负责运算并输出电流参考值，随后将此数值写入单片机内部的 DAC 模块转换为模拟基准电压。真实的电感高频电流信号被引入单片机内部高速模拟比较器的正端，负端接 DAC 的输出参考基准。比较器的翻转输出通过内部纯硬件逻辑直接连接至 PWM 模块的触发保护区。当电感电流冲高至设定阈值的瞬间，模拟比较器迅速翻转并直接通过硬件切断 PWM 信号。该流程全程无需 CPU 中断介入，兼具了数字算法处理外环的灵活性与纯硬件电流内环逐周期截断的极速响应能力。
