热设计基本概念与关键物理量
热设计的基础建立在热路与电路的类比关系之上。在电路中，电流流过电阻产生电压差；在热路中，功率损耗流过热阻产生温差。这构成了热设计的基础核心公式：

为了系统性地分析热传导过程，工程上将热学参量与电路物理量进行了等效映射：

电路物理量 热学物理量 物理本质对应关系
电流 热功率或损耗 () 发热源：MOSFET上的开关与导通损耗等同于源源不断注入的热电流
电压 温度 () 状态表现：节点电位高对应物体温度高，温差 等同于电压差
电阻 热阻 () 散热瓶颈：衡量物体或界面（如导热硅脂、散热片等）阻碍热量传递的能力，热阻越大热量越难散发
电容 热容 () 温度缓冲池：物体吸收热量但温度不上升的能力，由质量与比热容乘积决定，决定了器件对瞬时功率的响应速度
地 环境温度 () 最终归宿：所有的热流最终都要流向无限大的环境

在热传导路径中，有几个至关重要的温度与热阻参数，它们共同决定了系统的散热效能与可靠性界限：

关键参数 符号 定义与特性说明
结温 芯片内部PN结的实际工作温度，是评估器件可靠性的最关键参数，严禁超过规格书上限
壳温 器件封装表面的温度，是可通过热电偶或红外热像仪实际测量的物理参考点
环境温度 与器件进行热交换的直接环境流体温度，对于带散热器器件为周围空气温度，对于直接贴装芯片为周围电路板温度
结到外壳热阻 芯片内部到封装外壳的热阻，由芯片制造商决定，属于器件固有属性
外壳到散热器热阻 封装外壳到散热器之间的热阻，主要受导热界面材料性能和安装工艺影响
散热器到环境热阻 散热器到周围环境的热阻，受物理尺寸、空气流动状态、安装方向等显著影响，是最难精确确定的参数
结到环境总热阻 从芯片内部到周围环境完整散热路径上的总热阻

动态电-热等效网络与物理模型
仅依靠稳态计算公式 （相当于直流电路中的 ）在现代电力电子系统中存在严重局限。系统发热往往包含PWM高频脉冲开关损耗或电机起步瞬间的持续巨大浪涌。若用稳态模型计算持续 10μs 的 1000W 脉冲，会得出数千度结温的荒谬结论，与实际安全运行现象相悖。稳态计算失效的核心原因在于忽略了热容参量。

引入热容后，真实的芯片散热网络从静态热阻计算跨入动态 RC 阶梯网络。常见有两种核心模型：

  • Cauer网络：具有明确物理意义的等效模型。发热功率作为恒流源，依次流经代表各层级物理材料的 RC 并联模块，最终接入代表环境温度的恒压源。
  • Foster网络：器件数据手册中常提供的模型。由多个 RC 并联模块串联组成，虽在物理上不严格对应具体的材料层，但能实现一致的外部结温曲线高精度数学拟合。

在 Cauer 物理网络中，热阻与热容必须采用并联结构而非串联。当热流注入某一层材料（如铜基板）时，会同时发生两个动作：给热容充电导致自身升温，以及克服热阻向下一层更冷的结构传导。瞬间状态下，热流主要用于给热容充电，流过热阻极少；稳态状态下，热容充满电且温度稳定，热流全部从热阻通过。

工程上可将这种动态热路的缓冲作用形象地类比为三级水库模型：

  • 第一级水库对应硅晶圆自身：热容极小，仅需数毫秒即可充满，对下一层结构的传导热阻较大。
  • 第二级水库对应 DBC 陶瓷覆铜板与纯铜底板：热容较大，需数秒至十数秒充满，热阻表现为导热硅脂的阻值。
  • 第三级水库对应铝制水冷液冷板：热容极大，内部水冷液的流动带走热量相当于无限大热容，最终连接至环境温度。

散热路径设计与导热界面材料选择
为确保芯片产生的热量高效传递至外部环境，必须构建合理的散热路径并匹配适当的界面填充材料。

典型的硬件散热路径分类如下：

  • 带散热器的分立器件常见散热路径：
    • 路径一（性能优先）：开关管 → 导热硅脂 → 散热器 → 空气
    • 路径二（绝缘与性能兼顾）：开关管 → 导热硅脂 → 陶瓷片 → 导热硅脂 → 散热器 → 空气
    • 路径三（方便与绝缘兼顾）：开关管 → 导热绝缘垫片 → 散热器 → 空气
  • 电路板贴装器件（SMD）主要散热路径：
    • 热量从芯片封装底部的散热焊盘，经由焊料传导至电路板铜箔，再通过铜箔散发到周围空气。

导热界面材料专门用于填充固体接触面之间的微小空气间隙，以显著降低外壳到散热器的热阻。常见材料的特性对比如下：

材料类型 物理形态 综合特性分析
导热硅脂 膏状流体 热导率较高，能完美填充微小空隙，整体热阻极低；缺点是涂抹操作不便，且长期运行存在老化干涸风险
导热硅胶垫 固态弹性片 安装极其便捷，兼具电气绝缘性和物理缓冲弹性；缺点是热导率通常低于硅脂，接触热阻稍高
相变导热片 固态转半液态 常温呈固态便于贴装，在特定工作温度下软化呈半液态完美填充缝隙，兼顾了硅胶垫的便捷与硅脂的低热阻性能；缺点是物料成本较高
导热陶瓷片 固态硬片 常规材质为氧化铝或氮化铝，热导率极高且绝缘性能卓越；缺点是质地硬脆，对散热器表面平整度要求极高，且必须双面涂抹导热硅脂配合使用

工程仿真应用与热测量闭环验证
在工业级机电热联合仿真软件（如 PLECS）中，动态电-热网络展现出巨大的实战价值。通过输入器件手册提供的 Foster 热网络参数，软件能自动将电学损耗转化为注入热阻容网络的热流。

以电动汽车起步用的电机控制器为例，面对弹射起步数秒内高达 6000W 的发热脉冲，系统的动态热吸波效应如下：

  • 在 0 至 10 毫秒瞬间：巨量热流涌入，热容极小的第一级硅片水库迅速注满，结温急速飙升，热量开始向第二级铜底板溢出。
  • 在 0.1 秒至 3 秒加速期：热流持续注入。由于第二级铜底板的热容非常大，这 3 秒内产生的绝大部分热量都被用于给铜板升温。在第 3 秒末尾，硅芯片的结温可能刚好触及 165℃ 至 170℃ 的安全红线边缘。
  • 第 3.1 秒起步结束：发热功率回落至 1000W 的巡航水平。危机解除，巨大的铜板中囤积的热量开始缓慢通过尚未显著升温的第三级水冷板排走。
  • 若脱离动态热容吸波效应，仅按 3 倍峰值功率结合稳态热阻进行设计，必将导致逆变器及散热系统的体积与重量严重超标。

在物理样机阶段，必须通过专用的热测量工具进行闭环验证：

  • 热电偶：用于精确获取器件壳温（）的单点绝对数据。操作时必须使用导热胶等方式将探头尖端牢固且紧密地固定在器件表面。
  • 红外热像仪：用于直观获取整个电路板的宏观热分布情况，优势在于能极快定位异常热点区域。操作时需特别注意，目标物体表面材料的发射率参数设置会直接影响最终测温精度。

关于背面开窗加强散热的设计，需要注意阻焊油墨的红外辐射发射率大约在0.85至0.90之间，而纯裸铜的发射率极低，大约只有0.04。如果仅仅是开窗露出裸铜而不做任何后续处理，由于红外辐射能力的丧失，芯片的散热效果反而会变差。背面开窗真正起作用的正确处理方式包含两点：

  • 在开窗裸露的铜皮上手工或波峰焊堆上厚厚的焊锡，以增加热容和对流的表面积。
  • 配合导热垫使用，开窗是为了让导热硅胶垫没有阻焊油墨的阻隔，直接贴在铜面上将热量传导给金属外壳。

传热学物理机制

在热力学中，基尔霍夫热辐射定律规定在热平衡状态下，一个物体的辐射率即发射率 等于它的吸收率 。理想黑体吸收所有的电磁辐射，即 ，其向外辐射热量的能力也是最强的，即 。高端铝制散热器通常会做黑色阳极氧化处理，使表面发射率从0.05至0.1大幅提升到0.85至0.95。

对流与辐射的散热规律受不同公式支配。牛顿冷却定律定义了对流散热为 ，其中 是表面传热系数， 是表面温度， 是环境温度，呈现一次方的线性关系。斯特番-玻尔兹曼定律定义了辐射散热为 ，其中 是斯特番-玻尔兹曼常数，呈现四次方关系。因为辐射散热与绝对温度的四次方成正比，在极高温下辐射散热会绝对碾压对流和传导，而在常温和低温下对流和传导占据主导。

在电力电子常见的 至 温度区间内，辐射的作用不容忽视。在有风扇吹的强制对流环境中，对流系数非常大，对流散热占比超过95%。但在无风扇的自然对流或密闭机壳条件下，空气静止导致对流系数极低。这种情况下，即使芯片只有 ，辐射散热占总散热量的比例也可高达25%至40%。在被动散热设计中，必须利用高发射率的材质来增强这部分辐射散热能力。

材质、颜色与红外热辐射特性

电力电子设备工作温度下的对外辐射完全处于波长约8至10微米的远红外波段。在这个波段内，材质的物理属性比可见光颜色的影响更为决定性。塑料、橡胶、陶瓷、纸张、树脂等非金属材质在远红外波段几乎都是黑体，发射率和吸收率通常在0.85至0.95之间。光面的铝、铜等裸露金属在远红外波段的发射率和吸收率极低，仅约0.03至0.1，会直接反射热辐射。

关于金属外壳内壁贴绝缘纸的散热评估，需要结合辐射换热与传导热阻综合考量：

  • 纯裸金属外壳内壁会反射95%的红外辐射热，导致热量滞留并引发热烘烤效应。
  • 贴绝缘纸后，纸张的非金属特性使其具有极高的远红外吸收率，能完美吸收芯片辐射的热量。
  • 绝缘纸导热率极低，且背胶层与金属间的微小空气隙会产生巨大的接触热阻。热量被纸吸收后无法传导至外壳，导致绝缘纸表面温度急剧升高，温差消失后辐射换热彻底停止，整体散热效果反而下降。

兼顾绝缘与辐射散热的工程处理建议：

  • 针对铝壳，首选内壁阳极氧化处理。氧化铝呈非金属陶瓷特性，极薄且绝缘，远红外吸收率在0.85以上，热量可瞬间传导至铝基材。
  • 在铁、钢或铝壳内壁喷涂薄层绝缘树脂漆或烤漆，通过漆面的非金属特性提高吸收率，同时其传导热阻远小于绝缘纸。
  • 若受限成本需贴片，应使用高导热系数的绝缘硅胶布，同时满足高吸收率和快速传热的要求。

PCB阻焊油墨的颜色选择同样受远红外波段特性支配。各类颜色的油墨主体成分均为环氧树脂，因此在远红外波段下的发射率几乎完全一致，均在0.85至0.90之间，辐射散热能力没有区别。在实际工程中，黑色PCB因吸收可见光，会导致走线隐形不便于检修排查，且在UV曝光固化时易导致底层固化不完全影响良率。黑色PCB通常仅用于具有严格光学需求的LED显示屏防反光场景，或高度集成全靠设备测试的消费电子产品中以追求视觉美观。常规电力电子项目推荐使用绿油工艺，其成熟度高、对比度清晰，且红外辐射散热性能与其他颜色一致。
