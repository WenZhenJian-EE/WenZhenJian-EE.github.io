别拿低频思维调板子：高频与低频电路的根本区别

我们在实验室做板子，最怕的就是把低频那一套“只要导线连通就万事大吉”的思维直接套在高频电路上。

低频和高频的根本分界线，在于信号波长 和电路物理尺寸 能不能放在同一个量级来比。
波长大家都在高中背过：（其中 是电磁波在介质里的传播速度， 是频率）。

  • 低频电路：如果电路的物理尺寸远小于信号波长（比如 ），那整条导线上每个点的电压、电流在同一瞬间基本可以看作是完全一样的。这时候，我们用基尔霍夫定律、欧姆定律，用集总参数模型就能搞定。
  • 高频电路：一旦信号频率高了，波长缩短到可以和板子尺寸甚至走线长度相比拟时，我们就必须切换到分布参数模型（传输线模型）。这时候走线上的寄生电感、寄生电容效应会瞬间飙升，一段普通的铜皮也会变成复杂的“传输线”。导线上不同位置、不同瞬间的电压电流完全不一样，像波浪一样在走线上流淌。

下表是我们做方案时，用来警示开发团队的高低频思维红线对比：

对比维度 低频电路 (集总参数) 高频电路 (分布参数)
物理模型 理想导线，阻抗忽略不计 分布式电容电感（传输线）
电压/电流特性 导线各点电压电流瞬间一致 随着空间位置和时间呈波浪式变化
核心关注点 电压、电流（算静态压降和通流） 功率与阻抗匹配（防反射，求最大传输）
测量仪器 万用表、普通示波器 射频功率计、频谱仪、矢量网络分析仪 (VNA)
设计目标 算对压降，确保驱动能力够强 实现最大功率传输或维持信号完整性 (SI)

寄生天线效应：谁把噪声发射出去了？

大家都知道，交变电流流过导体就会向外辐射电磁波。而当导体的长度达到信号波长的四分之一（）时，就会发生强烈的谐振，这根导体就变成了一根极其高效的辐射天线。

辐射效率并不是一个“开或关”的临界点，而是一个连续渐变的过程：

  • 导体长度远小于 ：辐射效率极低，我们在工程上可以假定它不辐射。
  • 导体长度接近 ，…：谐振爆发，辐射效率升到天花板。
  • 导体长度大于 ：辐射效率依然很高。

[!WARNING]
我们在研发中碰到的 EMI 恶梦：
别以为只有高频信号线才会辐射。一个看似低频的网络（比如复位线、电源地线），如果它在空间上离高频骚扰源（比如 Buck 的 SW 节点）太近，骚扰源高频变化的电磁场会通过空间电场或磁场耦合，把噪声灌进这个低频网络。
如果这根低频导线、PCB 铜皮或者芯片引脚的物理长度恰好符合了该噪声频率的 谐振长度，它就会自发充当代言天线，把高频噪声高效地辐射出去，让你的 EMI 测试当场挂掉。
治本方法：拉开骚扰源和敏感导线的空间距离，因为电磁场能量是随着距离呈平方级衰减的。

撕开阻抗的假面：特性阻抗与直流电阻的本质区别

在高频电路里，我们嘴边挂得最多的词就是“阻抗”。

  • 输入阻抗：从电路输入端看进去的等效阻抗。
  • 输出阻抗：从电路输出端看进去的等效阻抗。
  • 阻抗不连续：如果传输线的走线宽度突然变窄、或者过孔打多了，导致特性阻抗突变，就会形成阻抗不连续点，这是信号反射的罪魁祸首。

1. 什么是特性阻抗 ？

特性阻抗是传输线自己娘胎里带出来的属性，由它的物理结构（走线宽度、走线到参考地的距离、介质介电常数）唯一决定，跟传输线的长度没有任何关系。它描述的是高频信号沿走线传播时，任意一点的瞬时电压和瞬时电流的比值：

注意：特性阻抗并不是个会发热消耗能量的电阻，它只是电磁波在介质中传播时感受到的“阻力”。

2. 特性阻抗 ↔ 直流电阻的物理对立

物理量 物理本质 决定因素 对信号的影响
特性阻抗 高频交流概念，描述介质能量传播的阻力 走线几何截面与介质，与长度无关 阻抗不匹配会导致信号发生反射与畸变
直流电阻 直流/低频概念，描述铜皮发热损耗的阻力 材料阻抗率、截面积以及走线长度 会把信号的能量转成热量，导致电压幅度衰减

3. 既然特性阻抗跟长度无关，为什么高频走线还要尽可能短？

经常有新人问我：“既然走线 阻抗做好了，拉 10cm 和拉 1m 特性阻抗都一样，为什么设计规范还要求越短越好？”
因为线拉长了，会触发其他致命问题：

  • 高频衰减加剧：虽然阻抗没变，但长走线的直流电阻和介质损耗会把高频信号的能量磨掉，传到终点时信号可能已经衰减到芯片认不出来了。
  • 信号延迟变大：电磁波传播速度是有限的（在 FR4 板材里大约是光速的一半）。线太长，延迟就大。在对时序要求极高的高速 DDR 或 SPI 总线里，延迟会直接导致采样错位。
  • 变成更好的天线：导线越长，越容易从周围环境拾取高频杂散干扰，信噪比直接垮掉。

阻抗匹配：我们在调板子时的两种目的

阻抗匹配的本质，就是为了防止高频信号在阻抗突变点发生反射，避免反射波和原始信号叠加在一起产生过冲、振铃。

我们在做阻抗匹配时，必须搞清楚你的目的是什么：

1. 最大功率传输匹配（射频 RF 思维）

  • 目标：要把源端的能量一分不剩地全送给天线。
  • 条件：负载阻抗必须等于源端阻抗的共轭（纯电阻电路里就是 ）。
  • 场景：射频天线、天线放大器匹配。能量本身就是生命线，必须追求效率最大化。

2. 信号完整性匹配（高速数字信号 SI 思维）

  • 目标：消除反射，保护数字波形的边沿，防止逻辑 ‘1’ 和 ‘0’ 发生误判。
  • 条件：消除反射波，不 care 功率传输效率，只要求波形干净。
  • 场景：高速 MCU 走线、SPI、DDR、CAN/RS485 总线。

[!NOTE]
我的研发心得：
数字通信走线，我们只在乎电压电平的逻辑对不对，不需要传递大能量；而射频电路，差一个分贝的功率都是致命的。

阻抗连续性控制：别在路上埋坑

在高速电路里（比如从 PHY 芯片 -> PCB 走线 -> 网络变压器 -> 连接器 -> 电缆 -> 接收端），我们设计的核心思想是阻抗连续性控制，而不是多级阻抗匹配。

这要求信号所经过的每一个环节，特性阻抗都要死死卡在同一个值（比如差分 ）。电磁波在这条阻抗绝对均匀的“高速公路”上跑，一路上碰不到任何阻抗不连续点，平稳跑到终点被终端电阻完全吸收。信号幅度的减小完全是走线物理损耗导致的，而不是被各级反射波折腾掉的。

我们在画高速板时，每一次打过孔、每一次走线拐直角，都是在给信号制造阻抗突变坑，必须严格按照规范控制。

降服反射：数字电路的信号完整性端接技术

如果阻抗没有控好，信号在接收端和源端之间来回反射叠加，就会在示波器上看到明显的过冲（Overshoot）和振铃（Ringing）。这会导致芯片时序错乱，或者引发严重的 EMI 辐射。

我们主要靠两种端接方式来降服反射：

1. 源端串联端接：先分压，再传播

  • 做法：在发送端引脚旁边紧挨着串一个电阻 。使得发送端输出阻抗与该电阻之和等于传输线阻抗：。
  • 物理过程：
    1. 信号发射瞬间，源端总阻抗和传输线 对驱动电压进行平分。进入传输线的电压波只有完整幅值的一半（半幅信号）。
    2. 半幅信号跑到末端接收芯片（高阻态开路），发生全额同相反射。反射波和入射波叠加，电压瞬间翻倍，恢复成完整的逻辑电平。
    3. 反射波往回跑，回到发送端时，刚好碰到匹配好的 阻抗，反射能量被这个组合完全吸收，不再发生二次反射。
  • 特点：只有在传输线最末端电压才是完整的；最爽的是没有静态功耗。

2. 末端并联端接：先传播，再吸收

  • 做法：在接收芯片引脚旁边，并联一个等于传输线阻抗的电阻到地或 VCC：。
  • 物理过程：发送端直接送出完整幅值的电压波。信号一路跑到末端，因为并联电阻和传输线阻抗完全匹配，信号能量在末端被电阻一滴不剩地吸收掉，根本不发生任何反射。
  • 特点：传输线全程波形都是干净、完整的；缺点是有静态功耗，发送端必须有极强的持续电流输出能力来喂饱这个终端电阻。

[!IMPORTANT]
串联端接和并联端接千万别混着设计：
如果你在源端串了电阻，又在末端并了电阻。根据分压，末端并联电阻会把电压拉死，接收端电压永远只有驱动电压的一半，系统直接锁死。
另外，菊花链总线的中间节点绝对不能放终端电阻，那会把信号能量在路口截胡，后续芯片直接收不到信号。

两大端接技术对比

我们在开发中一般按此表来选择端接方案：

特性 源端串联端接 末端并联端接
主要目标 吸收回来的反射波 吸收入射波，防反射
对驱动器的要求 普通驱动（单片机 GPIO, FPGA IO 就能带） 必须是强力专用驱动（RS485, CAN 收发器）
静态功耗 无 有（发热明显）
信号有效区间 仅在末端有效 在走线全程都有效
适用拓扑 点对点 或 单发送多接收 多点双向通信总线
典型应用 MCU 到 SPI Flash、主控到外设 RS485, CAN, LVDS, 工业以太网

终极博弈：信号完整性 (SI)、电源完整性 (PI) 与 EMC

我们在分析整机 EMC 问题时，SI、PI 和 EMC 三者其实是缠绕在一起的。它们的始作俑者都是功率开关管切换产生的 和 。

  • 信号完整性 (SI)：
    核心是解决阻抗连续性。SI 差导致的信号反射、振铃，会产生额外的 噪声。这种噪声能量主要被锁在信号走线和回流地构成的环路内部，表现为差模辐射。
  • 电源完整性 (PI)：
    核心是降低电源分配网络（PDN）的高频阻抗，抑制高频电源噪声。芯片高频开关时，会疯狂从电源轨抽取瞬态电流 。如果在 PDN 阻抗上产生电压跳变 ，这个共模电压噪声会顺着板子走线、排线、外壳发射出去，形成极其顽固的共模辐射。

在 EMC 整改中，我们常说一句话：做好 PI 是为了封杀共模辐射的源头，做好 SI 则是为了剿灭差模辐射的根基。
