硬件系统级接地与 ESD 瞬态电荷平衡的底层逻辑

搞硬件的都知道，接地和地线设计是个大坑。在直流或者低频电路里，地线也就是个零电位参考点；但只要频率一高，遇到静电（ESD）或者雷击浪涌，地线立马就变成了一个由寄生电感和电磁场主导的战场。

之前做过不少车载系统和工业仪器的开发，被地线问题折腾过好几回。这里就聊聊 ESD 放电的瞬态电荷平衡、浮地单板和电脑调试时的地电位反噬，以及我们经常挂在嘴边的“大地”，在工程上到底扮演着什么角色。

1. 浮地系统也会被静电打坏？聊聊 ESD 的物理本质

很多刚入行的新手有个直觉：觉得完全“浮地”（Floating）的系统（像电池供电的手持设备、无人机单板）反正没有接大地，静电没处泄放，所以不怕静电打坏。
这个直觉非常危险。

人体是个天然的高压静电储存电容，在干燥季节，人体电容 ，静电电压轻松攒到几千伏甚至十几 kV。而完全浮地的 PCB，它对大地也是有寄生电容的，只不过通常只有几 pF 到几十 pF。

当带静电的手指去碰浮地单板时，其实是一个高压大电容向一个低压小电容进行瞬态电荷转移。
电荷一过去，两个电容并联，单板的参考地 GND 电位在纳秒级的时间里会被瞬间拉高几千伏：

既然单板 GND 整体电位都被拉高了，为什么芯片还会被打坏呢？
因为电荷在 PCB 走线上蔓延是要花时间的。碰触瞬间，接触点的电压瞬间飙升，而 PCB 地线铜皮都是有寄生电感（ESL，大概 ）的。在高频瞬态极高的 冲击下，地线电感在局部会形成巨大的瞬态压降：

结果就是，电荷根本来不及均匀分布到整个 GND 平面上。挨打的管脚和附近的芯片 GND 引脚之间就会产生一个致命的瞬态绝对压差，瞬间把 MCU 内部的 ESD 钳位管或者栅极氧化层给穿透，芯片就这么废了。

  • 防雷设计：所以浮地系统也必须在接口和按键处放 TVS 管。而且 TVS 管的地引脚必须以最短的距离、最宽的线/铜箔连到芯片的主地平面，提供一个极低 ESL 的放电通道，在局部就把电压差消灭掉。

2. 调试浮地单板时的“地电位反噬”惨案

我们经常在实验室里调试电池供电的浮地板子。板子本身没接大地，但我们得用 USB 线或者 JTAG 仿真器把它连到台式电脑上。这时候注意了：台式电脑的三芯电源线是可靠接大地的（电脑 GND 连着 PE）。

这时候如果测试人员用带静电的手去摸板子上的传感器外壳：

  1. 如果板子是浮空的，没有连电脑，静电在板子上瞬态电荷转移，一般坏不了。
  2. 但现在插着 USB 线连着电脑。单板的 GND 瞬间被拉高到几千伏，而电脑主板的 GND 被保护地（PE）牢牢按在 0V。
  3. 瞬间，板子和电脑之间形成了一个巨大的压差。静电电荷在纳秒内，顺着 USB 的信号线或者地线，疯狂向电脑的 PE 通道倾泻。
  4. 结果就是，在这条泄放路径上的 USB 转串口芯片、主控 MCU，甚至电脑的 USB 接口直接被冲垮烧毁。我就亲眼见过因为这个把电脑主板南桥干废的。

怎么防范：

  • 总线隔离：在调试或者工业现场，敏感的通信总线（CAN、RS485、SPI 等）能加隔离的尽量加数字隔离芯片（比如 ADuM 系列或者隔离光耦），从物理上把两条地线阻断。
  • 接口防护：在 USB 接口这类连接器端，一定要配专门的 ESD 钳位阵列。另外屏蔽外壳（Shield）地和系统信号地之间，并一个 的高压电容和兆欧级大电阻，起到阻抗缓冲和高频泄放的作用。

3. 大地在工程上的两副面孔

我们在做系统接地设计时，“大地”在不同场景下的表现是完全不一样的：

3.1 面对瞬态电荷：无限大电容

对于 ESD 或者雷击浪涌这种瞬态、有限的电荷，地球是个庞然大物，半径 6400 公里，等效电容大约 。
当几千伏的静电（电荷量微乎其微，几微库）流进大地时，地球表面的电位变动连 1 毫伏都不到。这时候，大地就像一个防线深不见底的蓄水池，可以无条件吸收电荷，所以我们把大地电位定为绝对参考零电位（0V）。

3.2 面对持续共模干扰：低阻抗回流平面

但是在开关电源漏电、或者大功率共模干扰等持续有源电流的情况下，电荷绝对不会在泥土里凭空消失。根据基尔霍夫电流定律，电流必须流回源端，形成闭合回路。

  • 回流路径：漏进地线 PE 的电流，其实是把大地当成导体，一路流回几十公里外配电变压器中性点的接地桩，再流回变压器线圈。
  • PE 线才是关键：在供配电的 TN-S 系统里，设备漏电会触发漏保或者空开跳闸，不是因为泥土导电性能好（其实泥土的接地电阻有几个欧姆，阻抗很大，电流根本不够触发空开）。真正的短路大电流是顺着纯铜的 PE 线（保护地线）以极低阻抗跑回变压器的。PE 线是保命的，泥土不是。
  • 防脏地反噬：工业现场的大地往往流淌着各种大功率设备的漏电流和噪声，也就是常说的“脏地”。如果弱电敏感仪器的参考地直接和脏地死死连在一起，共模噪声就会反向灌进系统。这时候，在弱电输入侧加共模电感、磁环或者磁珠，把脏地的干扰阻断开，是非常有必要的。
