飞跨电容多电平逆变器 FC-MLI

飞跨电容多电平逆变器 FC-MLI

FC-MLI 是一种常用于中高压、大功率 DC-AC 变换的多电平逆变器拓扑，与中性点钳位 NPC、级联H桥 CHB 属于同类级别的经典架构。

  • 核心原理：利用悬浮（飞跨）在开关器件两端的电容进行工作。通过控制不同开关管的开通与关断组合，利用飞跨电容对母线电压进行分压和钳位，在输出端合成出阶梯状的交流电压波形，如三电平或五电平等。
  • 拓扑优势：电平数越多，输出的阶梯波越接近正弦波，谐波畸变率 THD 极低，可显著减小输出滤波器的体积。相较于需要多个独立直流电源的 CHB，FC-MLI 仅需一个公共直流母线，无变压器隔离需求。此外，对于同一输出电平，系统通常具备多种开关组合状态的冗余性，用于在运行中对飞跨电容进行充放电，维持电压自然均衡。
  • 工程挑战：随着电平数增加，高压薄膜飞跨电容数量呈几何级数增长，导致体积增大、成本升高，降低系统功率密度与可靠性。在死区时间、负载突变或低调制度等实际工况下，维持所有电容电压平衡属于高度复杂的闭环控制难题。系统上电时还需要专门电路或复杂软启动时序完成预充电。

开关电容变换器 SCC

SCC 是一种无磁元件的 DC-DC 变换拓扑，俗称电荷泵。其核心定位是实现低压、微型化、无磁件的极致高功率密度设计，旨在取代大体积磁性元件。

  • 核心原理：摒弃传统电感或变压器，依靠开关管网络重构，使电容在串联或并联等不同组合下充放电，实现电压倍增、分压或极性反转。
  • 拓扑优势：消除笨重的高频变压器与储能电感，实现极高的功率密度，可设计为极轻薄模块甚至完全集成于硅片上的 On-chip IC。由于无电感磁场泄漏，系统 EMI 表现优异。
  • 工程挑战：传统 SCC 仅能实现固定的电压转换比（如 2:1 或 1:3），宽范围稳压会导致极大效率损失。同时存在电荷分配损失，当两电压不同的电容通过开关硬连接时，会产生极大的瞬间浪涌电流，这部分能量损耗在开关管寄生电阻上，理论损耗公式为 。
  • 现代演进与应用：为消除硬开关浪涌电流损耗，现代设计采用谐振开关电容拓扑（如 STC 或 Dickson 拓扑），引入微小谐振电感实现开关管的 ZCS 或 ZVS 软开关。该技术在数据中心与 AI 服务器的 48V-12V 中间母线架构 IBA 中广泛应用，作为 4:1 固定变比直流变压器，效率可逼近 99%。在消费电子领域，100W 至 200W 的手机超快充核心降压级也大量使用高频 SCC 网络实现 4:1 或 2:1 降压，满足手机内部严苛的空间限制。

双有源桥变换器 DAB

DAB 是处理中大功率（如 150kW PCS、30kW DC-DC）、高压及需要电气隔离场景的核心前沿拓扑。当工程目标涉及千瓦级以上功率传输、要求隔离或宽范围闭环稳压时，DAB 是首选方案。在实际工程中，DAB 的设计难点主要集中在复杂软开关 ZVS 区间划分、死区时间优化、变压器偏磁消除（如占空比微调或隔直电容选型）以及并联架构下的均流控制。

飞跨电容三电平 Boost 变换器基本概念

飞跨电容三电平 Boost 变换器（FC 3L-Boost）是一种经典的多电平拓扑结构。其中，飞跨电容（）是该拓扑的核心元件。其被称为飞跨电容的原因在于它的电位状态：该电容的两端均未接地，也未连接到固定的电源母线上，而是完全悬浮在电路中间。它的一端连接在两个二极管（D1 和 D2）的中间节点，另一端连接在两个开关管（T1 和 T2）的中间节点。当 T1 和 T2 处于不同的开关状态时， 两端的绝对对地电压会剧烈跳变，如同在电路节点之间飞跃。同时，它跨接在两个半桥桥臂的中点，因此得名。

开关模态分析
在正常的交错控制下，T1 和 T2 的 PWM 驱动信号永远相差 180°（即半个开关周期）。假设稳态时输出电压为 ，飞跨电容 的电压被完美控制在 ，整个电路存在 4 种基础开关模态。

1 T1 开通，T2 开通 -> L -> T1 -> T2 -> GND 旁路（保持不变） 输入电源直接给电感 L 充电，电感承受正向电压 。D1 和 D2 承受反压截止。

在实际运行中，这 4 个模态的组合由占空比 D 的大小严格决定：

  • 重度升压工况（）：由于 T1 和 T2 的导通时间有重叠，模态循环为 模态 1 -> 模态 2 -> 模态 1 -> 模态 3。电路永远不会进入模态 4。
  • 轻度升压工况（）：由于 T1 和 T2 的导通时间较短，中间会出现都不导通的真空期，模态循环为 模态 4 -> 模态 3 -> 模态 4 -> 模态 2。电路永远不会进入模态 1。

飞跨电容电压均衡原理
无论占空比如何变化，在一个完整周期内，模态 2（给电容充电）和模态 3（给电容放电）的时间必须绝对相等。
根据电荷量公式 ，在这两个模态下，流过飞跨电容的电流均是电感电流 （短期内可视为恒定直流）。要使飞跨电容的电压保持在 ，其在一个周期内吸收的电荷必须等于释放的电荷。在电流相同的前提下，必须保证导通时间完全一致。

理想状态下，只要 T1 和 T2 的占空比相等且相位严格错开 180°，数学上即可保证模态 2 和模态 3 的时间相等，这被称为自然均压特性。但在工程实现中，由于驱动芯片延迟差异、MOSFET 或 IGBT 开关时间差异以及布线寄生电感不对称等因素，绝对的时间相等难以实现。如果模态 2 比模态 3 每个周期仅多导通 10 纳秒，飞跨电容的电压就会逐渐上升并最终导致器件损坏。因此，实际应用中必须引入闭环主动均压控制律强行修正电压偏差。

相位与占空比的概念辨析
相位与周期严格相关，本质上描述的是信号触发的时间先后，与占空比（脉冲的宽度）完全独立。例如，A 信号占空比 70%，B 信号占空比 50%，B 信号相位滞后 A 信号 50°。这表示 B 信号仅比 A 信号晚了周期 T 乘以 的时间出现，两者的脉宽和起始时间独立受控。

在多电平拓扑控制中，互补导通与相差 180° 相位（交错并联）存在本质区别：

控制策略 核心逻辑 占空比关系 底层实现 典型应用
互补导通 状态互斥（如 T1 开则 T2 关） 占空比之和等于 1（如 30% 与 70%） 共享同一个三角波发生器，占空比参考值分别取正相与反相输出 普通半桥（如 Buck 电路的上下管）

脱离了两个开关管对称工作的语境，任意相位差在工程中都有特定用途，其核心逻辑通常为 ：

  • 相差 90°（交错 4 相）：应用于大电流主板 CPU 供电的 VRM 模块（多相 Buck 电路），4 组 PWM 依次错开 90°，以实现输出电流的最优平滑。
  • 相差 120°（交错 3 相）：应用于工业三相电机驱动或 3 相交错 PFC，三个桥臂的触发时间依次错开 120°。
  • 任意角度移相（如相差 30°）：应用于 DAB（双有源桥）或移相全桥拓扑，原边与副边均输出 50% 占空比方波，通过设定副边波形滞后特定角度（如 30°），在变压器电感两端产生压差从而传递功率。

开关电容拓扑分析

核心物理与数学基石

在纯开关电容网络中，由等效电阻引起的总发热在物理本质上归属于电荷再分配损耗，学术界称为双电容悖论。此项损耗并非无功损耗，而是百分之百转化为热量或辐射的有功损耗。系统缺乏电感的连续能量平滑传递机制，本质属于离散采样系统，通过电容接力搬运电荷，其微观瞬态电流表现为高能脉冲。

技术演进与损耗消除

为打破纯开关电容网络中因极值电压差导致的硬开关宿命损耗，工业界与学术界发展出谐振开关电容变换器拓扑结构。

  • 拓扑物理重构：在常规的开关与电容充放电回路中，针对性地串联极微小的电感器，通常为数十纳亨级别，或直接复用 PCB 走线寄生电感。
  • 状态轨迹转移：此微小电感将原本暴烈的 RC 指数级大爆炸冲击电流，有效转化为平滑可控的 RLC 欠阻尼谐振状态，电流按正弦波规律柔性上升并回落。
  • 零电流开关控制：通过精准的高频控制逻辑，在电流正弦波刚好落回零值点，即半个谐振周期瞬间断开开关，实现完全软过渡。
  • 能量无损回收：在此拓扑下，原本由电荷再分配效应必定蒸发散失的 能量，在谐振半周期的前半段转存为电感磁场能，并在后半段由电感完整释入储能电容。该技术彻底消灭了宏观内阻极限，使系统纯导通损耗仅取决于真实的 ，成为高性能大功率主板电源的核心拓扑基石。

开关电容拓扑的阻抗分析法

开关电容拓扑的阻抗分析法

阻抗分析法是处理变压器结合开关电容拓扑（如倍压整流电路、电荷泵）的核心分析工具。由于开关电容电路存在不断的充放电瞬态过程，直接分析时域波形较为复杂。阻抗分析法能够将非线性、非定常的开关过程等效简化为线性、稳定的电阻网络，从而便于评估变压器绕组负担及选择MOSFET的电流余量。

倍压电路的一个典型问题是电压跌落现象，即负载电流越大，输出电压下降越明显。通过阻抗分析，可以将包含变压器与倍压电路的整体模型视为一个理想电压源串联内阻的模型：

  • 内阻1：由变压器等效漏感 产生的感抗。
  • 内阻2：由开关电容网络产生的等效电阻 。

在设计谐振拓扑并附加倍压级时，需确保上述内阻总和不会导致输出电压跌落至规格要求之下。
