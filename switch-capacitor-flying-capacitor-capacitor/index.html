飞跨电容多电平逆变器 FC-MLI

在做中高压、大功率 DC-AC 变换时，飞跨电容多电平逆变器（FC-MLI）可以说是我们经常碰到的经典架设了，跟NPC（中性点钳位）和级联H桥（CHB）算得上是同级别的老战友。

我把我们在研发中总结的核心要点简单梳理一下：

  • 核心工作机制：它的核心思路就是让电容“悬浮”（飞跨）在开关器件两端。我们通过控制不同的开关管导通组合，利用这些悬浮电容对直流母线电压进行分压和钳位，最终在输出端拼出一个台阶状的交流电压波形（比如三电平或五电平）。
  • 我们在项目里选它的原因：电平数越多，输出的阶梯波就越贴近完美的正弦波，谐波畸变率（THD）能做到极低，这能帮我们省下大量滤波器的体积。跟需要一堆独立直流源的级联H桥相比，FC-MLI 只需要一个公共直流母线，而且不需要隔离变压器。最爽的是，对于同一个输出电平，系统里有很多种开关状态的冗余性，我们在控制算法里可以利用这些冗余状态给飞跨电容充放电，实现电压的自然均衡。
  • 我们踩过的坑和技术挑战：一旦电平数变多，高压薄膜飞跨电容的数量就会呈几何级数增加，电路板体积和成本直接飙升，可靠性也大打折扣。而且在死区时间、负载突变或者低调制度这些实际工况下，怎么稳住所有电容的电压不飘移，是个让人头秃的闭环控制难题。另外，上电时的预充电软启动时序要是没设计好，一不留神就会直接炸管。

开关电容变换器 SCC

再聊聊开关电容变换器（SCC），大伙在研发里习惯叫它电荷泵。我做低压、微型化设计时，如果想把体积缩到极致、彻底干掉大块头的电感变压器，首先想到的就是它。

  • 核心工作机制：它直接摒弃了传统的电感或变压器，全靠开关管网络去高频重构电容的连接方式，让它们在串联或并联状态下切来切去地完成充放电，从而搞定电压的倍增、分压或极性反转。
  • 我们在项目里选它的原因：干掉了笨重的高频变压器和储能电感后，功率密度高得吓人，特别适合做极轻薄的电源模块，甚至能完全集成到硅片里做 On-chip IC。而且因为没有电感，不需要担心磁场泄漏，系统的 EMI 表现非常干净。
  • 我们在开发中踩过的坑：传统的 SCC 只能实现固定的电压转换比（比如 2:1 或 1:3），一旦想做宽范围稳压，效率会崩得没法看。而且它存在不可避免的“电荷分配损失”——当两组电压不同的电容通过开关硬连接时，会产生极大的瞬间浪涌电流，这部分能量全损耗在开关管和走线的寄生电阻上了，损耗公式就是经典的 。
  • 现在的技术演进与应用：为了干掉这部分硬开关的浪涌损耗，我们在实际研发里会引入微小的谐振电感（比如几十 nH 的贴片电感，或者直接复用 PCB 走线寄生电感），搞成谐振开关电容拓扑（如 STC 或 Dickson），实现开关管的 ZCS 或 ZVS 软开关。这技术现在在数据中心和 AI 服务器的 48V-12V 中间母线架构（IBA）里火得不行，作为 4:1 的固定变比直流变压器，效率能硬生生逼近 99%。在消费电子领域，大伙用的 100W 到 200W 手机超快充，内部核心降压级也全靠这种高频 SCC 网络做 4:1 或 2:1 降压，不然手机里根本塞不下。

双有源桥变换器 DAB

当我们的工程目标涉及千瓦级以上功率传输、要求隔离或者宽范围闭环稳压时，DAB 是首选方案。在实际工程中，DAB 的设计难点主要集中在复杂软开关 ZVS 区间划分、死区时间优化、变压器偏磁消除（如占空比微调或隔直电容选型）以及并联架构下的均流控制。

飞跨电容三电平 Boost 变换器基本概念

我们经常在升压设计里碰到飞跨电容三电平 Boost 变换器（FC 3L-Boost）。在这个拓扑里，飞跨电容（）就是绝对的灵魂。大家之所以管它叫“飞跨”或“悬浮”电容，是因为它的电位在工作时是完全悬空的：它的两端既不接大地，也不接固定的直流母线，而是完全漂浮在电路中间。一端接在两个二极管（D1 和 D2）的中间，另一端接在两个开关管（T1 和 T2）的中间。当 T1 和 T2 高频开关时， 两端对地的绝对电压会产生剧烈跳变，就像在电路节点之间飞跃一样，而且它又是跨接在两个半桥桥臂中点的，所以大伙形象地叫它飞跨电容。

开关模态分析

我们在做交错控制时，T1 和 T2 的 PWM 驱动信号永远相差 180°（即半个开关周期）。假设稳态下输出电压为 ，飞跨电容 的电压被完美稳在了 ，整个电路工作时会有 4 种基础开关模态：

模态 开关状态 电流路径 飞跨电容（）状态 说明
1 T1 开通，T2 开通 -> L -> T1 -> T2 -> GND 旁路（保持不变） 输入电源直接给电感 L 充电，电感承受正向电压 。D1 和 D2 承受反压截止。
2 T1 关断，T2 开通 -> L -> D2 -> -> T2 -> GND 充电 电感释放能量。电流穿过 D2，自上而下流过 ，通过 T2 流回地。 吸收电感能量。
3 T1 开通，T2 关断 -> L -> T1 -> -> D1 -> 放电 电感电流经 T1 流至 下极板，自下而上穿过 ，顶开 D1 流向输出端。 将模态 2 吸收的能量与电感能量一同释放给负载。
4 T1 关断，T2 关断 -> L -> D2 -> D1 -> 旁路（保持不变） 两管全关，电感电流直接通过两个串联的二极管流向输出端。

我们在实际调试时，这 4 个模态的组合是由占空比 D 的大小决定：

  • 重度升压工况（）：T1 和 T2 的导通时间有重叠，此时模态循环为：模态 1 -> 模态 2 -> 模态 1 -> 模态 3，电路永远不会进入模态 4。
  • **轻度升压工况（）**：T1 和 T2 的导通时间比较短，中间会出现都不导通的真空期，模态循环为：模态 4 -> 模态 3 -> 模态 4 -> 模态 2，电路永远不会进入模态 1。

飞跨电容电压均衡原理

不管占空比怎么变，在一个完整周期内，模态 2（给电容充电）和模态 3（给电容放电）的时间必须绝对相等。

根据电荷量公式 ，在这两个模态下，流过飞跨电容的电流均是电感电流 。要使飞跨电容的电压稳在 ，它在一个周期内吸进去的电荷必须等于吐出来的电荷。在电流相同的前提下，我们必须保证这两个模态的导通时间完全一致。

理想状态下，只要 T1 和 T2 的占空比完全相等，且相位严格错开 180°，数学上就能实现自然均压。但在我们实际调试中，因为驱动芯片的延迟、MOSFET 的开关开关时间差异、PCB 布线寄生电感不对称等工程因素，绝对的导通时间相等是不存在的。哪怕每个周期模态 2 比模态 3 多导通了 10 纳秒，时间一长飞跨电容电压也会漂走，最后导致器件过压烧毁。所以，我们在工程实现里必须引入闭环主动均压算法来强行微调修正这个偏差。

相位与占空比的概念辨析

相位是描述信号触发的时间先后，与占空比（脉冲宽度）完全是两码事。比如，A 信号占空比 70%，B 信号占空比 50%，B 信号相位滞后 A 信号 50°。这只代表 B 信号比 A 信号晚了特定时间才起来，两个的脉宽和起始时间完全是独立受控的。

在多电平拓扑控制中，“互补导通”和“相差 180° 相位（交错并联）”有着本质区别：

控制策略 核心逻辑 占空比关系 底层实现 典型应用
互补导通 状态互斥（如 T1 开则 T2 关） 占空比之和等于 1（如 30% 与 70%） 共享同一个三角波发生器，占空比参考值分别取正相与反相输出 普通半桥（如 Buck 电路的上下管）
交错并联（相差 180°） 动作复制，延时触发 占空比完全相等（如均为 40%） 运行两个独立的三角波发生器，频率相同但在时间轴上错开 180°，即错开 飞跨电容 Boost 等需要均流或均压的并联拓扑

除了交错180°，在多相并联或者移相拓扑中，其他的相位差也大有用处：

  • 相差 90°（交错 4 相）：常见于主板 CPU 供电的 VRM 模块（多相 Buck），4组 PWM 依次错开 90° 触发，能把输出电流纹波压到极小。
  • 相差 120°（交错 3 相）：常见于三相电机驱动或者三相交错 PFC，三个桥臂依次错开 120° 触发。
  • 任意角度移相（如相差 30°）：在双有源桥（DAB）或者移相全桥拓扑里，原边和副边都输出 50% 占空比的方波，通过控制副边波形滞后原边特定角度，在变压器漏感两端产生压差来传递功率。

  • 48V转12V纯开关电容变换器：为了满足服务器或车载大电流供电，用 Dickson 等拓扑做固定比例降压。彻底不用电感，靠优化开关频率 和电容 把等效内阻压到毫欧级，实现极高的功率密度。
  • 芯片内部电荷泵设计：集成电路内部为了产生负压或高编程电压，用微小电容搭电荷泵。IC 设计工程师靠等效内阻公式精确规划硅片面积，保证带载时输出电压不掉下来。

技术演进与损耗消除

为了打破开关电容拓扑中硬开关电荷再分配必然带来发热损耗的“宿命”，我们在设计中通常会用谐振开关电容变换器：

  • 物理结构重构：在充放电回路中串联一个极小的谐振电感（几十 nH，有时直接复用 PCB 走线的寄生电感）。
  • 状态轨迹转移：这个微小的电感能把原本暴烈的 RC 指数级冲击电流，柔化为 RLC 欠阻尼谐振，电流呈正弦波规律温和升降。
  • 零电流开关控制：在电流正弦波刚好落回零的瞬间（半个谐振周期）断开开关，实现完美的 ZCS 软开关。
  • 能量无损回收：原本在硬开关下必定转化为热量散失的 能量，在谐振前半周转存为电感磁场能，后半周由电感完整释放给了储能电容。这个技术消灭了慢开关极限的内阻，让系统损耗只取决于实际的 。

隔离倍压整流电路仿真规范

我们在用软件做这个电路的仿真时，经常有兄弟踩中下面这几个导致波形异常的坑：

问题类型 现象与影响 原理分析 修正方案
破坏电气隔离 副边参考点强接到原边隔直电容中间节点，导致原副边波形错乱。 原副边本应是独立的隔离电路。共地会把副边倍压电路的充放电电流直接灌入原边的交流耦合节点。 删除连接原边和副边的共地线，给副边电路单独连接本地参考电位。
带载过重 负载电阻设置过小，输出电压被瞬间拉垮。 倍压整流电路的能量传输依赖飞跨电容。在 20kHz 开关频率下，1μF 电容的容抗约为 8Ω。接入极小负载会由于串联分压导致负载端电压极低，并可能引发极大的无功电流。 将负载电阻改为合理的轻载范围，如 1000Ω 或 10000Ω。
仿真精度不足 仿真输出电压比实物偏高。 理想脉冲源忽略了驱动电路的压降；理想二极管忽略了正向导通压降。高频倍压整流中通常使用肖特基二极管或快恢复二极管。 考虑原边供电的实际峰值，并将副边二极管的正向压降设置为肖特基二极管的实际值，如 0.4V。

开关电容拓扑的阻抗分析法

我们在开发开关电容拓扑或者倍压整流时，阻抗分析法是不可或缺的武器。因为电路一直在快速充放电，直接分析时域波形会让人抓狂。阻抗分析法能把非线性的开关换流等效成线性电阻网络，方便我们评估变压器的绕组负荷以及选择开关管的电流余量。

电荷泵倍压电路最典型的毛病就是“带载电压跌落”，我们在设计谐振拓扑加倍压级时，要把变压器等效漏感感抗和开关电容网络等效电阻 算在一起，确保总内阻不会把输出电压拉到规格以下。
