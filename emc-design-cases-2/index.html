输入电压84V系统EMI设计

系统的EMI滤波电路设计

针对输入电压可能达到 84V、整机电流预估在 2A 左右的系统，推荐采用 TVS 结合 Pi 型滤波架构。此设计利用共模电感自身的漏感充当差模电感，无需额外增加差模电感器件。

电路连接顺序与拓扑：
从双电源二极管汇集点至后级输入端，各元件依序连接如下：

  • 输入防浪涌节点：在前端母线与地之间并联 TVS 管。
  • 第一级滤波：在前端母线与地之间依次并联大容量铝电解电容与高频 MLCC 电容。
  • 共模抑制：串接共模电感，其输入侧引脚连接前端母线与地，输出侧引脚连接后级母线与隔离地。
  • 第二级滤波：在共模电感输出侧的后级母线与隔离地之间并联 MLCC 电容。
  • Y电容：若系统外壳存在物理接地，在共模电感输出侧分别并联安规电容至外壳大地。

标准化器件参数清单：

模块位置 器件类型 推荐参数 封装与材质 功能描述
防浪涌保护 TVS SMDJ85A 或 SMDJ85CA / 85V / 3000W SMC 并联于输入端，吸收电机反电动势等低频大波动。
第一级滤波 铝电解电容 220uF / 100V / ±20% 10x10mm 贴片或插件 吸收浪涌并储能，必须选用高频低阻型。
第一级滤波 MLCC 电容 1uF / 100V / ±10% C1206 / X7R 滤除中频开关噪声。为保证直流偏压特性，避免使用极小封装。
第一级滤波 MLCC 电容 0.1uF / 100V / ±10% C0805 / X7R 滤除高频尖峰干扰。
共模抑制 共模电感 1mH / 3A 1211 贴片或立式磁环 阻断高频噪声。必须选用大尺寸确保满载运行不饱和、不发热。
第二级滤波 MLCC 电容 1uF / 100V / ±10% C1206 / X7R 滤除经电感泄漏的中频残留噪声。
第二级滤波 MLCC 电容 0.1uF / 100V / ±10% C0805 / X7R 滤除经电感泄漏的高频残留噪声。
Y电容滤波 安规电容 2.2nF / 1000V / ±10% C1206 / X7R 泄放共模干扰至大地。容值须控制在 1nF 至 4.7nF 之间以符合安规漏电流要求。

浪涌保护与瞬态响应分析

在最高电压达到 84V 的系统中，合理评估瞬态过压保护器件与滤波电容的协同工作特性对于系统可靠性至关重要。

元件动作时间与耐压特性：
硅雪崩型 TVS 的内部响应时间为皮秒级，能够迅速应对微秒级或纳秒级的浪涌尖峰。对于标称连续直流工作电压为 100V 的 MLCC 电容，其介质击穿电压极高，通常可承受数倍额定电压的极短脉冲。

钳位电压的影响与电解电容的缓冲作用：
以 SMDJ85CA 为例，其工作电压 为 85V，击穿电压 在 94.4V 至 104V 之间。面对极限浪涌电流，其最大钳位电压 可能达到 137V 左右。这种瞬时高压可能会对 100V 耐压的 MLCC 及后级芯片造成威胁。
第一级滤波中的 220uF 大容量电解电容在此处提供关键的缓冲作用。根据公式 ，电容两端的电压不能突变。浪涌脉冲的能量会优先为该大容量电容充电，从而减缓电压上升速率，将尖锐的高压尖刺转化为平缓的波动。该机制有效降低了 TVS 实际面对的峰值电流，使系统实际面临的钳位电压通常维持在 100V 左右。

系统耐压优化方案：

  • 提升输入电容耐压规格：为确保应对钳位电压的高余量，可将第一级滤波的 MLCC 替换为更高耐压规格，如采用 1uF 250V C1210 规格替代原 100V C1206 规格，采用 0.1uF 250V C1206 规格替代原 100V C0805 规格。
  • 确认核心芯片极限耐压：必须核对后级降压 DC/DC 芯片的输入引脚极限耐压规格，确保其具备承受 100V 甚至更高瞬态电压的能力。

PCB布线规范与避坑指南

硬件电路的实际效能与安全高度依赖于 PCB 的布线设计，布局布线需严格遵循以下准则。

  • 瞬态保护首当其冲：TVS 管必须放置在电源线刚刚进入电路板的接插件或相关焊盘的最近端。外部浪涌进入系统后必须最先被 TVS 吸收，严禁通过过长走线连接 TVS 器件。
  • 强制滤波走线顺序：电源网络布线必须严格依照电流流入方向，依次穿过大容量电解电容、中容量 MLCC 和小容量 MLCC 的焊盘，最后进入共模电感。禁止使用打孔飞线方式进行简单并联。
  • 核心器件区域禁布：共模电感正下方的所有 PCB 铜层必须设置为禁布区，绝不允许铺设地铜皮或横穿其他信号线。内部层铺铜会产生寄生电容，导致高频噪声直接越过电感，从而导致滤波功能失效。
  • 修改网络名称以强制布线：建议在原理图中经过共模电感后改变线路网络名称，以强制要求布线时网络必须穿过电感，防止铺铜时发生意外的直接短接。
  • 噪声源就近接地泄放：Y电容必须放置在共模电感的输出侧，即靠近系统高频噪声源的一侧。产生的高频共模干扰会优先通过 Y电容流向外壳大地，残留噪声再被高阻抗的共模电感阻隔，从而避免噪声向输入端线缆泄漏并产生辐射。Y电容连接车架大地的走线必须尽可能短且粗，建议就近打过孔连接至固定金属外壳的螺丝孔环网络。
